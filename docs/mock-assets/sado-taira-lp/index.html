防衛道路アクセス

航空自衛隊佐渡分屯基地の管理道路を抜けて辿り着く、全国でも類を見ない入山儀式。赤い電話ボックスから自衛隊へ連絡し、許可を得てゲレンデへ——秘境感そのものが体験価値。
