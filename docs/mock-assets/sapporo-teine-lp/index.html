温活・回復

アフタースキー、
温泉と食で満たす。

豊平峡の雪見露天とインドカレーのカオス、手稲直下のほのか、小樽IC直結のオスパ——ウィンタースポーツ後の疲労回復を多層的に設計。

タトゥー対応の貸切風呂は湯元小金湯（90分¥3,800〜）。プライバシーを重視する欧米系旅行者の強力なフック。
