**周辺の主要競合ゲレンデの特徴**
備北エリアには、異なる強みを持つ複数の競合が存在します。これらとの差別化がマーケティング上の急務です。
*   **スノーリゾート猫山（庄原市）:** 道後山の山頂からも目視できる近隣のライバルです [cite: 15]。ピラミッド型の巨大センターハウスを持ち、人工降雪機（スノーガン・ICS）を完備。安定したコンディションを提供し、若年層のスノーボーダーに強いのが特徴です [cite: 18, 19, 20]。
*   **ひろしま県民の森スキー場（庄原市）:** 道後山と同じく天然雪100%主体で、ブナ林の樹氷など自然環境に優れます。リフト2本・4コースを備え、中上級スキーヤー向けの斜面や自然志向の層からの評価が高いです [cite: 14, 21, 22, 23]。
*   **りんご今日話国スキー場（庄原市高野町）:** 滑走距離450m・3コースの非常に小規模な施設ですが、ソリ遊びなど雪遊びに特化しており、小さな子連れのファミリー層を完全にターゲットにしています [cite: 14, 24]。
*   **芸北高原大佐スキー場（北広島町）:** 芸北エリアを代表する施設。国道沿いでアクセスが良く、幅400mの広い緩斜面など全7コースを備え、初心者やファミリー向けに絶大な人気を誇ります（※過去ドラフトでの言及にあった「芸北国際 / やわたハイランド191」などの大型リゾート群とも顧客を二分しています） [cite: 25, 26, 27]。

**競合に対する「優位性」と「劣後要因」**
競合環境における道後山高原スキー場の立ち位置は以下の通りです。
*   **優位性:** 芸北高原大佐などの大型リゾートと比較した際の「圧倒的な低価格」と「混雑のなさ（リフト待ちゼロ）」です [cite: 15, 25, 27]。また、猫山のような人工雪メインのゲレンデと比較すると、降雪時の「パウダースノーの質（天然雪の感触）」と、山頂からの「中国山地のパノラマ景観」では頭一つ抜けています [cite: 2, 5, 14]。
*   **劣後要因:** 最大の劣後は「営業の不安定さ」です。猫山などが人工降雪機でオープン日を確約できるのに対し、道後山は直前まで営業可否が読めません。さらに、最新のセンターハウスや動く雪道（エスカレーター）などを備える近代的なゲレンデと比較すると、設備面での不便さ（特に徒歩での登りや旧式リフト）が顕著に劣っています [cite: 3, 12, 15, 27]。

## 4. ターゲット層のインサイト（口コミ分析）

SNSや各種スキー場レビューサイトから顧客の生の声であるUGC（User Generated Content：ユーザー生成コンテンツ）を抽出し、顕在的・潜在的なニーズと不満点を分析します。

**口コミから読み解く顧客体験の実態**
レビューデータからは、当スキー場に対する評価が「絶賛」と「困惑」の二極化傾向にあることが浮き彫りとなりました。

*   **ポジティブな傾向（絶賛の声）:**
    最も多い称賛は「雪質」に関するものです。「チョーパウダー」「また来ます！さいこー！」といった、広島県内とは思えない雪質に感動する声が多数見受けられます [cite: 8, 12, 16]。また、「20年以上ぶりに訪れたが、コースを滑ると懐かしくなった」「古いスキー場ならではの絶景が見られる」といった、昭和レトロな雰囲気やロケーション自体を楽しむ（エモさを感じる）声も特徴的です [cite: 15, 17]。
*   **ネガティブな傾向（不満と課題）:**
    不満のほぼ全てが「設備とインフラ」に集中しています。「下のリフトは長くて良いが、上部にもリフトがあるのにかなりの登りなのでとても大変」「動く歩道を設置するか、上はクローズでも良いのでは」という、物理的な疲労に対する切実な声が記録されています [cite: 4, 8, 12]。また、「コースマップがどこにあるか分からない」「駐車場の除雪が大胆（不十分）でスタックが怖い」といった、ホスピタリティや運営のソフト面に対する改善要求も見逃せません [cite: 15]。

**インサイトの統合：顕在化していない隠れたニーズ**
これらの口コミから導き出される隠れたニーズは、「不便さを上回る『納得感』の提供」です。利用者は設備の古さ自体には必ずしも怒っておらず、事前の情報不足（歩く必要があると知らされていなかった等）に不満を抱いています。逆に言えば、このレトロさや不便さを「秘境ゲレンデならではのアドベンチャー」として事前にブランディングし、期待値をコントロールできれば、不満を面白さに転換できる余地が存在しています。

## 5. 周辺観光資源とのシナジー（エリア回遊性）

スキー場単体での売上（リフト券・ゲレ食）には限界があります。庄原市西城町という立地を活かし、周辺の温泉やグルメと連携した「エリア回遊ルート」を構築することで、滞在時間を延ばし、地域全体での消費単価を上げることが不可欠です。

**近隣の有望な観光・グルメ資源**
道後山周辺には、スキー後の疲労回復や食欲を満たす強力なコンテンツが密接に点在しています。

*   **温泉施設:**
    *   **ひばごん郷温泉 すずらんの湯:** スキー場から車で約5〜6分、「ひば道後山高原荘」に隣接する日帰り温泉施設（大人620円） [cite: 2, 28, 29, 30]。源泉かけ流しでサウナや水風呂も完備しており、スキー客の鉄板ルートとなっています [cite: 10, 31]。
    *   **高尾の湯 / 比婆山温泉 熊野湯:** さらに足を伸ばせば、pH11の強アルカリ性（皮脂や古い角質を溶かし、肌をツルツルにする効果がある）の秘湯「高尾の湯」や、昭和レトロな一軒宿「熊野湯」などのディープな温泉体験も可能です [cite: 32, 33, 34]。
*   **ご当地グルメ:**
    *   **ドライブインミッキー:** 庄原IC近くにある創業55年の老舗定食屋。デカ盛りメニューや牛乳瓶での提供など、SNS映えする要素を持ち、常に賑わっています [cite: 35, 36, 37]。
    *   **和食もりおか:** 庄原産牛肉と特産「つくね芋」を使った「牛とろ定食」が人気の本格和食店です [cite: 38]。
    *   **つちのこ邸:** 大正時代の古民家を改装した西城町のカフェ。1日10食限定の定食が人気で、今後はサウナの併設も予定されており、感度の高い若年層の回遊拠点になり得ます [cite: 39, 40]。
    *   **一寸そば屋 / 備後西城駅 EKINAKA:** 地元の在来種そば粉を使った本格的な手打ち蕎麦が楽しめます。特にEKINAKAは鉄道ファンにも人気です [cite: 41, 42, 43]。

**売上倍増のための「回遊ルート構築案」とロジスティクス**
これらの資源を組み合わせ、ターゲット別のパッケージを提案します。以下は具体的な移動時間と1人あたりの推定費用（概算）を伴うルート案です。

1.  **「極上の雪と湯・サウナ」ととのいルート（若年層ボード層向け）:**
    *   *ロジスティクス:* スキー場出発 → (車移動：約6分) → 「すずらんの湯」で入浴・サウナ → (車移動：約40分) → 庄原IC付近の「ドライブインミッキー」で夕食。
    *   *1人あたり推定費用:* 約5,120円（リフト券3,500円 ＋ 温泉620円 ＋ 夕食約1,000円）
    *   *狙い:* 満足度MAXの満腹ルートを提供し、日帰り客の地元消費を促す。
2.  **「昭和ノスタルジーと秘湯」エモ旅ルート（シニア・カップル向け）:**
    *   *ロジスティクス:* ゲレンデ食堂で昼食後出発 → (車移動：約15分) → 秘湯「高尾の湯」または「熊野湯」で入浴 → (車移動：約30分) → 「和食もりおか」で夕食。
    *   *1人あたり推定費用:* 約6,000円（リフト券3,500円 ＋ 温泉代 ＋ 夕食約1,500〜2,000円）
    *   *狙い:* 心と体を癒すプレミアムな体験を提供し、シニア層の客単価を引き上げる。

スキー場の公式WEBサイトやSNSでこれらの「アフタースキーの過ごし方」を積極的に発信し、地元店舗への送客を図りつつ、スキー場への来場動機を「雪山＋α」に引き上げることが重要です。

## 6. 最新トレンドの適応可能性

気候変動による雪不足リスクを抱える当スキー場が生き残るためには、冬のスキー営業に依存しないビジネスモデルの構築や、最新トレンドの取り込みが必須です。

**通年化（グリーンシーズンの活用）**
道後山高原スキー場にとって最もポテンシャルが高いのが「通年化」です。夏の平均気温は21度と冷涼であり、避暑地として最適です [cite: 6]。
*   **キャンプ / グランピング:** 広いゲレンデと無料の広大な駐車場は、オートキャンプ場や星空観賞イベントの会場として即転用可能です（近隣のりんご今日話国スキー場は夏キャンプで成功しています） [cite: 9, 44]。
*   **クロスカントリーと合宿需要:** 隣接する「道後山高原クロカンパーク」は、春から秋にかけて陸上競技の実業団や学生の合宿地として有名です（58haの敷地に専用芝コースを整備） [cite: 45, 46, 47]。このスポーツ需要を取り込み、ゲレンデをマウンテンバイク（MTB）のダウンヒルコースや、トレイルランニングの大会会場として貸し出すことで、グリーンシーズンの安定収益を確保できます。

**インバウンドの取り込み余地**
現在、庄原エリアのスキー場における訪日外国人客（インバウンド）の比率は極めて低い状況です。しかし、ニセコや白馬の過度なリゾート化（インバウンド価格の高騰）に疲れ、日本の「ローカルで本物の文化」を求めるリピーター層（欧米豪のコアスキーヤー）は確実に存在しています。
道後山は「100%天然雪のパウダー」「昭和レトロな食堂」「本物のローカル温泉（すずらんの湯）」が揃っており、「Authentic Japan Ski（古き良き日本のスキー）」として海外向けにニッチなPRを行えば、マニアックな層を呼び込む余地は十分にあります。
*   *成功のケーススタディ:* 例えば、長野県や東北地方の小規模ローカルゲレンデ（例：斑尾高原など）は、大規模なリゾート開発を行わずとも、「豊富なパウダー」と「昔ながらの日本の原風景（ローカルな温泉街など）」をセットで発信することで、ニセコの混雑を避ける外国人客の誘致に成功しています。道後山もこのモデルを踏襲可能です。

**地形再整備による必要積雪量低減（物理的アプローチ）**
莫大な投資が必要な人工降雪機の導入は現実的ではないため、オフシーズン中にゲレンデの草刈り、石拾い、地形の平滑化（土木作業）を徹底的に行う必要があります。
*   *ロジスティクス:* 地元の山岳会やファンを巻き込んだ「ボランティア整備イベント」を開催するほか、庄原市などの過疎地域向け観光インフラ整備補助金を活用し、夏期にスタッフと小型重機を集中的に投入します。これにより、「積雪20cmでも板が傷つかずに滑れる」平滑な下地を作り出し、天然雪ゲレンデの弱点である「オープンに必要な積雪量」のハードルを下げ、物理的に営業日数を増加させます。

**新規軸（DX、サステナビリティ）の導入余地**
すでにレンタルショップでは各種キャッシュレス決済が導入されていますが [cite: 6]、リフト券のオンライン事前決済や、公式LINEを用いた「今日の積雪・パウダー情報」のリアルタイム配信（プッシュ通知）を導入することで、降雪時の機動的な集客が可能になります。また、隣接する「ひば道後山高原荘」にWi-Fi環境を整備し、「午前中はパウダーを滑り、午後は温泉に入りながらテレワーク」という冬季ワーケーションプランを造成することも有効です [cite: 29, 45]。
