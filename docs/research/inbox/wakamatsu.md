## Executive Summary
当レポートは、北見若松市民スキー場の売上倍増と平日ナイター稼働向上を目指し、LP構築用データ要件を完全に満たした実行計画と包括的調査結果をまとめたものである。
*   **立地の優位性:** 北見市街地から車で約15分という圧倒的な近距離に位置し、日常的なレクリエーション施設としてのポテンシャルが極めて高い [cite: 1, 4, 5]。
*   **ナイター環境の質:** 眼下に北見市街の夜景を望む景観と、朝夕2回の徹底した圧雪による高品質なバーンが強みである [cite: 5, 6, 7]。
*   **最大の課題:** 1月〜2月のナイター時間帯（夜間）における平均気温が氷点下10度〜13度（平均最低気温は氷点下14度前後）という極端な寒冷気候と [cite: 8, 9, 10]、リフト2基（ペア1基・ロープトウ1基）という構成に対する料金の「割高感」が、平日夜間のリピート率低下を招いている [cite: 2, 7, 11, 12]。
*   **成長戦略の核:** 市街地至近という立地と、北見市特有の「焼肉文化」や温泉資源をシームレスに結合し、運転代行やタクシーを活用した合法的なロジスティクスを構築することで、アフタースキーを含めた「平日夜のマイクロツーリズム」を実現する [cite: 13, 14, 15]。併せて、他施設の成功事例に基づくサウナ併設等の高付加価値化と時間単位サブスクリプションの導入により、単価と来場頻度の最大化を図る。

北見若松市民スキー場は、北海道北見市が運営する地域密着型の都市型スキー場である [cite: 4, 12]。本レポートは、市街地至近かつナイター営業という同施設の特性を最大限に活かし、特に「平日夜の稼働率向上」を通じた売上倍増を実現するための深掘り調査と戦略的洞察をまとめたものである。提供された各種データおよび最新の口コミ、周辺観光資源の分析を統合し、施設単体の魅力向上に留まらない、地域全体を巻き込んだマーケティング戦略を提示する。

## 1. 基礎データと現状のポジショニング

当スキー場は、オホーツク圏の中心都市である北見市の市街地から至近に位置し、主に地元市民の練習場や学校教育の場として機能してきた [cite: 5]。ここでは、施設の基本的なスペックと現在の市場における立ち位置を整理する。

**基本情報および設備仕様**
当スキー場はコンパクトな敷地の中に、初心者から競技者まで対応するバリエーション豊かなコースと、ナイター設備を完備している。

*   **規模:** 総面積約22ha、標高差176m（山頂294m〜318m、ベース142m〜143m） [cite: 2, 16]。
*   **索道（リフト）設備:**
    *   アカシアリフト（ペアリフト・2人乗り）: 全長693m。山麓のスキーセンター正面から山頂までを直接結ぶメインリフト [cite: 1, 17, 18]。
    *   ロープトウ（スーパーA線）: 全長210m。初心者や子供向けの練習用として機能している [cite: 1, 17, 18]。
*   **ゲレンデ・コース構成:**
    *   Aコース（中級）: 全長1,080m、最大斜度20度、平均斜度9度。第2リフトからアクセスするメインのロングランコース [cite: 3, 5, 19]。
    *   Bコース（上級）: 全長693m、最大斜度23度、平均斜度14度。SAJ（全日本スキー連盟：日本におけるスキー競技を統括する団体）公認コースであり、競技者のトレーニングにも使用される [cite: 3, 5, 19]。
    *   Cコース（初・中級）: 全長900m、最大斜度20度、平均斜度11度 [cite: 3, 19]。
    *   その他: ファミリーゲレンデ（平均斜度10度の緩斜面）、ちびっこ広場（そりコース） [cite: 5, 18]。
*   **料金体系（一般大人）:** 1日券3,000円〜3,900円、4時間券2,200円〜2,800円、ナイター券（1時間）700円〜1,000円。ロープトウ1日券1,100円。シーズン券33,000円。※過去データや時期によって変動がある [cite: 2, 7, 11, 18]。
*   **アクセス:** JR石北本線北見駅から車（タクシー）やバスで約15〜20分。駐車場約500〜700台（無料） [cite: 2, 4, 7, 19, 20]。

**現在の市場におけるターゲット層と立ち位置**
現在の主なターゲットは、北見市内に居住するファミリー層、部活動やスキースクールで技術向上を目指す学生、および仕事帰りの社会人である [cite: 4, 5]。大規模リゾートのような非日常性を提供する施設ではなく、仕事や学校の終わりにふらっと立ち寄れる「日常の延長線上にある高規格な練習場」という独自のポジショニングを確立している [cite: 5, 21]。特に人工降雪機と朝夕2回の圧雪によるバーンコンディションの良さは、基礎スキーヤーやアルペン競技者から高く評価されている [cite: 1, 6, 7]。

## 2. 強みと弱み (内部環境)

施設の内部環境を分析することで、売上倍増に向けたレバレッジポイントと、顧客体験を損ねているボトルネックを明らかにする。

**強み (Strengths)**
当施設の集客の最大の武器は、アクセスの良さと雪面の管理品質である。

*   **市街地からの圧倒的な近さ:** 北見駅から約15分という立地は、平日夜間の集客において最大の武器である。仕事帰りや学校帰りに気軽に立ち寄れる環境は、他の山岳リゾートにはない強みである [cite: 1, 4, 5]。
*   **徹底したバーン整備と雪質:** 人工降雪機による下地作りと、ナイター営業前を含む朝夕2回の圧雪作業により、常にエッジが噛む引き締まったベストコンディションが維持されている [cite: 1, 6, 7]。
*   **充実したナイター環境と景観:** 山頂までナイターで滑走可能であり、眼下に広がる北見市街の美しい夜景は、都市型スキー場ならではの魅力的な景観資源である [cite: 5, 6]。

**弱み (Weaknesses)**
一方で、気候条件や施設規模に関する構造的な弱みが、リピーター獲得の障壁となっている。

*   **極端な寒冷気候:** オホーツク圏特有の厳しい寒さは雪質を良好に保つ反面、1月〜2月の平均最低気温は氷点下13度〜14.2度に達し、ナイター時間帯（夜間）の平均気温も氷点下10度〜13度で推移するという極端な寒冷気候（極寒）である [cite: 8, 9, 10]。この「とにかく寒い」という顧客の不満は、夜間の滞在時間を短縮させる直接的な要因となっている [cite: 12]。
*   **施設規模と料金のミスマッチ（割高感）:** 実質的にメインのリフトが1基（アカシアリフト）のみであり、コースバリエーションが限られるにもかかわらず、ナイター1時間で700円〜1,000円という料金設定に対し、一部のユーザーから「北海道のローカルゲレンデとしては高い」との指摘が出ている [cite: 2, 11, 17, 18]。
*   **強風による運行停止リスク:** 気象条件によってはリフトの運行が中止されるケースがあり、利用者の計画的な来場を阻害する不確実性を含んでいる [cite: 3, 17]。

内部環境を総括すると、当スキー場は「技術を磨きたい」という明確な目的を持つ層には最適な環境を提供する一方で、短時間で寒さを感じやすく、価格に見合う満足感を得られにくいという課題を抱えている。平日夜の稼働率を上げるためには、この「寒さ」と「割高感」をいかに払拭し、付加価値を提供するかが急務である。

**優位性と劣後要因**
これらの競合に対する当スキー場の立ち位置を分析する。

*   **優位性:** 北見市街地（繁華街やビジネス街）からの物理的な近さにおいて、ノーザンアークリゾート（車で約20〜40分）や八方台スキー場（車で約40分）を凌駕している。また、SAJ公認コースを有するなど、斜面変化と滑りごたえ、バーン整備の質では若松市民スキー場が上回っており、本格的な練習環境として圧倒的な優位性を持つ [cite: 5, 13, 19, 23, 26]。
*   **劣後要因:** ノーザンアークリゾートはホテルや温泉が直結しており、アフタースキーを含めた「リゾート体験」のパッケージ化が容易である。対して若松は、休養村センター（宿泊・入浴可能）は隣接しているものの [cite: 7, 29]、温泉や飲食における非日常的なリゾート感では一歩譲る。さらに、競合が低価格路線（ノーザンアークのナイター2,000円や八方台の2時間1,500円等）を打ち出している場合、時間当たりの相対的な価格競争力の低下が懸念される [cite: 13, 18, 23, 26]。

競合との差別化を図るためには、リゾート感で勝負するのではなく、**「市街地至近」という機動力を活かした都市連動型のマイクロツーリズム**へと舵を切る戦略が有効である。

## 4. ターゲット層のインサイト（口コミ分析）

Web上の口コミやレビューから、実際の利用者が抱いている顕在的・潜在的なインサイトを抽出する。

**ポジティブ・ネガティブな傾向**
顧客の声は、ゲレンデの質に対する高い評価と、環境や価格に対する不満にはっきりと二分されている。

*   **ポジティブな声:** 「丁寧な圧雪がされており、カービングターンを楽しむには良いゲレンデ」「バーンの整備がとても良い」など、斜面のコンディション維持に対する賛辞が多い [cite: 11, 12]。アルペンスキーヤーや基礎スキーヤーにとって、質の高い練習ができる場として信頼されている。
*   **ネガティブな声:** 「バーンは最高！とにかく寒い！」「北海道のローカルゲレンデでナイター1時間700円は少し高く感じる。2時間700円なら文句なし」といった声が見受けられる [cite: 11, 12]。

**隠れたニーズと不満点（インサイト）**
ここから読み取れるターゲットのインサイトは、**「短時間で集中して滑りたいが、寒さの限界が早く訪れるため、時間単位のチケット価格に対して滑走量が釣り合わず、結果として割高に感じる」**という心理構造である。
つまり、顧客は単に値下げを求めているのではなく、「寒さに耐えられる時間内で、満足のいく滑走体験とお得感」を求めている。この不満を解消するためには、単なる時間券の値下げではなく、「短時間利用のサブスクリプション化」や、滑走後の「暖を取る体験（サウナ、温泉、焼肉等）」とのセット化が求められている。

## 5. 周辺観光資源とのシナジー（エリア回遊性）

平日夜の稼働率とエリア全体の消費単価を上げるためには、スキー場単体で完結させず、北見市の強力な周辺資源とのシナジーを創出する「回遊ルート」の構築が不可欠である。

**近隣の有望な観光・飲食資源**
北見市周辺は、独自の食文化と温泉資源に恵まれている。

*   **北見焼肉（ソウルフード）:** 北見市は人口当たりの焼肉店数が北海道内で最も多い「焼肉の街」である。かつて家畜処理場があった歴史から、新鮮な内臓肉（豚ホルモンや牛サガリ）を無下味のまま七輪で焼き、特製の「生ダレ」で食べるスタイルが市民のソウルフードとして定着している [cite: 15, 30, 31]。市街地には「四条ホルモン」「焼肉道食」「百萬力」「味覚園 総本店」など名店がひしめき合っている [cite: 30, 32, 33, 34, 35, 36]。
*   **周辺温泉:** 市街地や近郊には、日帰り入浴が可能な温泉施設が多数存在する。留辺蘂地区には「北見温泉（ポンユ）三光荘」「滝の湯センター夢風泉」「塩別つるつる温泉」など、源泉掛け流しで肌に良いとされる名湯が多く、極寒の中で冷え切った体を温めるには最適な環境である [cite: 37, 38, 39, 40, 41, 42]。
*   **その他スポット:** 山の水族館（北の大地の水族館）、北見ハッカ記念館、アドヴィックス常呂カーリングホールなど、日中の観光資源も豊富である [cite: 14, 43, 44, 45]。

**回遊ルート構築案（平日夜・マイクロツーリズム）**
平日夜間のスキー場利用者をターゲットとした回遊ルートを提案する。

*   **「極上の圧雪＆極上の赤身」北見ナイター満喫ルート:**
    18:00 退社後、北見若松市民スキー場へ直行（車で15分）。
    18:30 ナイターチケット（1時間〜1.5時間）で、完璧に圧雪されたBコースを数本集中してカービング滑走。夜景を堪能。
    20:00 スキー場を出発し、市街地の焼肉店（例：味覚園や百萬力など）へ移動。
    20:20 冷えた体を七輪の熱気で温めながら、新鮮な牛サガリと生ビールでアフタースキーを満喫 [cite: 13, 15, 31]。

**※ロジスティクスと安全性の確保（重要）:** このシナリオを成立させるためには、自家用車での移動と飲酒という相反する要素を解決しなければならない。飲酒運転を完全に排除するため、北見市内に多数存在する「運転代行サービス」の利用推進やタクシーの積極的活用、あるいは主要な宿泊施設（ルートイン等）と連携し、スキー場と市街地の焼肉店を結ぶ「専用シャトルバス」の運行といった、現実的かつ合法的な交通ロジスティクスを事前に構築・明記することが不可欠である。

## 6. 最新トレンドの適応可能性

社会的な最新トレンドを導入することで、従来のアプローチでは獲得できなかった新規顧客層を開拓する。

**インバウンド需要の取り込み**
北見市は女満別空港から車で約40分という立地にあり、網走の流氷観光や知床方面へのハブ都市として、インバウンドを含む観光客の経由地となっている [cite: 21, 46, 47]。現在、インバウンド客が夜間に滞在する際、飲食以外のナイトライフコンテンツが不足している。スキー用具を一式レンタル可能な当施設を「日本のローカルな都市型ナイターと夜景が楽しめるスポット」としてパッケージングし、近隣ホテルと連携して送迎付きプランを提供することで、新たな収益源となる。

**通年化（グリーンシーズンの活用）**
当スキー場は、オホーツク企画株式会社が管理する「若松キャンプ場」や「きたみファミリーランド」と隣接している [cite: 2, 7, 29]。夏季には広大なゲレンデ斜面を活用したマウンテンバイク（MTB）のダウンヒルコースの造成や、夜間の星空観賞・夜景ツアーを実施することで、通年での収益確保とスタッフの安定雇用が図れる。

**新規軸（DX・サステナビリティ・ワーケーション）**
「とにかく寒い」という弱点をテクノロジーで克服するアプローチが有効である。例えば、事前予約・決済アプリの導入（DX）により、屋外のチケット売り場に並ぶ時間をゼロにする。また、麓の休憩所にスマートロック式の「貸切無煙サウナテント」や、リモートワークが可能な「ワーケーション用Wi-Fi完備ヒーティングラウンジ」を設置する。これにより、滑走しない同伴者の快適性を向上させ、滞在時間を飛躍的に伸ばすことができる。

## 結論：売上倍増に向けたアクションプラン

ここまでのリサーチとインサイトの分析に基づき、平日夜間の稼働率を劇的に向上させ、売上倍増を達成するための具体的なアクションプランを提案する。

**短期アクション（1年以内で実行可能・即効性重視）**
