# 三川温泉スキー場 売上倍増戦略レポート

## Executive Summary
新潟県阿賀町に位置する「三川・温泉スキー場」は、新潟市中心部から車で約1時間の圏内にありながら、最大斜度38度の非圧雪コースと、車で約10分圏内に広がる「三川温泉郷」という強力なシナジーを持つローカルリゾートである。ファミリースキー場としてのコストパフォーマンスと、上級者を満足させるマニアックな地形を併せ持つ特異なポジショニングを確立している。
しかし、現状の売上と持続可能性を脅かすコア・ボトルネックとして、①低標高（ベース121m）による頻繁な雪不足リスク、②リフトの間引き運転を常態化させている深刻な人員不足、③決済手数料やシステム・通信への懸念から脱却できない完全現金主義（アナログな運営体制）、の3点が挙げられる。
本レポートでは、これらの課題を克服し売上倍増を達成するためのアクションプランを提示する。短期的な施策としては、「通信・コスト課題をクリアした上でのQRコード決済・事前Webチケットの導入」により機会損失を防ぎ、近隣温泉との「デジタル周遊券創設」でエリア全体の客単価を向上させる。中長期的な施策としては、外部の観光DX補助金等を活用した「ICオートゲート導入による抜本的な省人化」と、グリーンシーズンのアウトドア資源化による「通年型リゾートへの脱皮」を強く提言する。

## 1. 基礎データと現状のポジショニング

三川・温泉スキー場は、五頭連峰を越えた大松平山の北西斜面に広がる中規模のローカルリゾートである。ファミリースキー場としての認知が高い一方で、上級者をも唸らせるハードなコースを併せ持つ特異なポジションを確立している。

### 基本情報および設備構成
当ゲレンデを構成する主要な諸元と設備・料金体系は以下の通りである。

*   **規模と標高**: 総面積23ha、トップ標高416m、ベース標高121m（標高差295m） [cite: 1, 2]。
*   **コース構成**: 全10コース（初級30％、中級40％、上級30％）。最長滑走距離は1,000m。最大斜度38度を誇る非圧雪のエキスパート向け「ダイナミックコース」や、初心者向けの広い緩斜面などを有する [cite: 3, 4, 5]。
*   **リフト設備**: ペアリフト4基 [cite: 3, 6]。ただし、近年の従業員不足により、日によって2基または3基のみの稼働となるケースが常態化している [cite: 7]。
*   **スノーボード比率**: スノーボード全面滑走可能（来場者比率：ボード60％、スキー40％） [cite: 3, 5, 6]。
*   **アクセス**: 磐越自動車道「三川IC」より約8km（車で約15分）。JR磐越西線「三川駅」よりタクシー・車で約5分と、インターからのアクセスが良好である [cite: 2, 8, 9]。駐車場は1,200台収容で全日無料開放されている [cite: 5, 6]。
*   **料金体系**: リフト1日券は大人3,500円、4時間券2,000円。レディース・シニア（50歳以上）2,500円、子供2,000円と、県内でも非常にリーズナブルな価格設定となっている [cite: 3, 5, 10]。

### ターゲット層と市場における立ち位置
現在の主要顧客層は、下越地方（新潟市や新発田市周辺）から訪れるファミリー層や初心者が中心である。「料金が安く、混雑が少ないため練習に最適」という認知が定着している [cite: 2]。同時に、地形を活かした最大斜度38度のオフピステ（非圧雪）コース目当てに来場する一部のコアな上級スキーヤー・ボーダー層も抱えており、二極化した顧客層を惹きつけているのが特徴である [cite: 1, 3, 5]。

## 2. 強みと弱み (内部環境)

売上向上戦略を立案する上で、自施設の強み（武器）を最大化し、弱み（ボトルネック）を無効化するアプローチが不可欠である。

### 強み (Strengths)
集客の強力なフックとなる内部要素は以下の通りである。

1.  **温泉リゾートとの物理的・心理的近接性**: スキー場の名称が示す通り、約6kmあまりの距離（車で約10分程度）に「三川温泉」および「新三川温泉」の温泉街が控えている。アフタースキーの温泉入浴（リフト券提示で入浴料100円引き等の連携あり）という強力な付加価値を提供できる [cite: 1, 2, 9, 11]。
2.  **独自の無料サービス（下部洗車）**: アクセス道路の路面に雪解け水を流しており、来場するだけで車の下部洗浄が無料でできるという、非常にユニークで実用的な特典がある [cite: 3, 5]。融雪剤による車両の塩害を気にするドライバーにとって隠れたアピールポイントである。
3.  **コースの多様性**: 小規模ながら10コースを擁し、初心者向けのワイドな緩斜面から、エキスパート向けの「ダイナミックコース（38度）」まで、地形を巧みに利用した滑り応えのあるレイアウトを実現している [cite: 1, 2, 6]。

### 弱み (Weaknesses)
一方で、顧客体験を著しく損ね、持続可能性を脅かしている課題が存在する。

1.  **低標高に起因する雪不足リスク**: ベース標高が121mと低いため、近年の暖冬傾向による雪不足の直撃を受けやすい。オープン延期や営業日数の減少（例：シーズン58日間のみの営業など）が頻発しており、安定した収益基盤の構築を阻害している [cite: 2, 7, 12]。
2.  **深刻な従業員不足**: スキー場運営においてマンパワー不足が顕在化している。スタッフ不足を理由に、土日であってもペアリフト全4基中2〜3基しか運行できない事態が発生しており、顧客の利便性低下を招いている [cite: 7]。
3.  **完全現金主義による摩擦と導入障壁**: 阿賀町の広報等でキャッシュレス決済導入の準備が報じられた時期もあったが、現状では「リフト券、レストラン、レンタルすべて現金払いのみ。クレジットカード・電子決済は使用不可」と明記されている [cite: 7, 13]。中小事業者や自治体施設においてキャッシュレス化が進まない理由として、決済手数料（3〜5%）による利益水準への負担感、入金サイクルの長さによる資金繰りへの影響、停電や端末故障等の非常時対応への不安、そして導入インセンティブの乏しさといったシステム面の課題が複合的に絡んでいると推測される [cite: 14, 15, 16]。しかし、このインフラの遅れは現代の消費者ニーズとの間に大きなギャップを生み、機会損失に直結している。

### 主要競合施設の詳細と優位性・劣後要因の統合分析

*   **NINOX SNOW PARK（ニノックス スノーパーク）**:
    新発田市に位置し、新潟市内から高速道路を使わずに新新バイパスと国道7号線経由で約60分、日本海東北自動車道「聖籠新発田IC」からは約40分という圧倒的な近接性を誇る [cite: 17, 18, 19]。コースは2本とコンパクトだが、最長22時までのロングナイター営業を実施している。特筆すべきは、「スマイル18（雪マジ世代応援プロジェクト）」というマーケティング施策である。これは18歳のリフト券を完全に無料にし、19歳〜22歳にも無料・半額割引を提供するもので、若年層を強烈に惹きつけている [cite: 17, 20]。また、ICリフト券システムも導入済みである [cite: 17]。
*   **国設胎内スキー場**:
    新潟市から車で約1時間（約50分）、日本海東北自動車道「中条IC」から約25分の距離にある下越地方最大のスキー場である [cite: 21, 22, 23]。全13〜14本のコースと、7基のリフト、スノーエスカレーターを擁し、初級から上級まで幅広い層を受け入れている [cite: 23, 24]。リフト1日券は一般5,000円であるが、「アソビュー！」等のプラットフォームを通じたWeb事前購入（割引クーポン付き）に対応しており、さらに毎週日曜日は「スキー子どもの日」として小学生以下のリフト料金を無料にするなど、強力な集客施策を展開している [cite: 23, 25]。

**統合分析**:
三川温泉スキー場の最大の**優位性**は、「圧倒的なコストパフォーマンス」と「温泉街とのシナジー」である。リフト1日券3,500円という価格設定は、ニノックスや胎内（大人5,000円）と比較しても安価であり、ファミリー層の財布に優しい [cite: 3, 17, 23]。また、アフタースキーとして車で約10分の「三川温泉」への動線は、他の都市型ゲレンデにはない情緒的価値を提供している [cite: 1]。
一方の**劣後要因**は、「雪の安定性」と「DX（デジタルトランスフォーメーション）の遅れ」である。ニノックスがICゲートや若者向けの戦略的割引でマーケティングを高度化させ、胎内がWeb事前購入を整備しているのに対し、三川は窓口での現金決済というアナログ運用に留まっており、顧客の利便性低下や購買単価向上（アップセル）の機会を完全に逸している。

## 4. ターゲット層のインサイト（口コミ分析）

SNSやレビューサイト、個人ブログにおける顧客の生の声を分析すると、施設側が認識している価値と、顧客が実際に感じている価値の交差点が見えてくる。

### ポジティブな傾向：昭和レトロな安心感と意外なハードさ
「こじんまりとしていて子供連れや初心者にオススメ」「料金が安く下越お勧めのゲレンデ」「混みすぎない雰囲気でのんびり滑って温泉で休みたい人に合う」といった声が主流であり、コストパフォーマンスの高さと混雑の少なさが最大の支持要因となっている [cite: 2, 26, 27]。一方で、上級者からは「最大斜度38度は伊達ではない」「非圧雪でコブもあり、滑りごたえが十分すぎる」と、ローカルゲレンデらしからぬコースの険しさがマニアックな評価を集めている [cite: 1, 3, 5]。

### ネガティブ・潜在的な不満：インフラとオペレーションへの懸念
隠れたニーズや不満点として顕著なのが、「リフトの稼働状況」と「雪質の悪化」である。「従業員不足でリフトが数本しか動いていなかった」という運営側の台所事情が顧客の目にも明らかになっており、体験価値の低下に直結している [cite: 7]。また、低標高ゆえに「水分を吸った重い雪になりやすい」「下部はコース幅が狭く、人が集まると危険」といった雪質・コースレイアウト上の安全性への懸念も見え隠れする [cite: 1]。

## 5. 周辺観光資源とのシナジー（エリア回遊性）

スキー場単体での集客・収益化には限界があるため、阿賀町全体の観光資源を巻き込んだ「エリア回遊性」を高める戦略が必須である。

### 有望な周辺観光資源
*   **温泉・宿泊施設**: スキー場から約6kmの距離にある三川温泉（にゅーゆもと、湯元館など）は、昭和の風情を残す名湯である。鯉の甘煮や山菜料理などの郷土料理が名物であり、スキー場利用者に「リフト1日券・4時間券提示で入浴料100円引き」といった連携を既に行っている [cite: 1, 9, 11, 28]。
*   **ご当地グルメ**: ゲレンデ食（ヒュッテ ビーウィンドのナポリタン、ふるさと山荘のラーメン等）に加えて、町内には「三川観光きのこ園」（冬季休業の可能性に留意しつつ、通年化への布石として重要）や、津川エリアの純手打ち蕎麦「塩屋 橘」、ジャンボ餃子で有名な「桃園楼」などの強力な食のコンテンツが存在する [cite: 29, 30, 31, 32]。
*   **道の駅 阿賀の里**: 車でアクセス可能な距離にあり、特産品の購入や食事（にぎわい亭）が可能。阿賀野川ライン舟下りの拠点でもある [cite: 33, 34]。

### エリア消費単価を上げる「回遊ルート構築案」
日帰りスキー客を「滞在型観光客」へと転換させる回遊ルートを提案する。
午前中は三川温泉スキー場で滑走（初心者は緩斜面、上級者はダイナミックコースを堪能）。昼食はゲレンデ内施設、あるいは少し足を延ばして津川エリアの「桃園楼」や「塩屋 橘」で地域特有のグルメを味わう。午後は再び雪遊びを満喫した後、車で約10分の三川温泉（にゅーゆもと等）で冷えた体を温める。帰路には「道の駅 阿賀の里」へ立ち寄り、阿賀町特産の日本酒（麒麟山など）や糀製品を土産として購入する。この一連のルートをパッケージ化して訴求することで、町内での一人当たり消費額（LTV）を劇的に向上させることが可能である。

## 6. 最新トレンドの適応可能性

社会構造の変化やテクノロジーの進化に合わせて、ゲレンデ運営のあり方をアップデートする必要がある。

*   **インバウンド (訪日外国人客)**: 現在のところインバウンド比率は無に等しいと推測されるが、「レトロなローカルスキー場 × 伝統的な日本旅館（鯉料理と源泉掛け流し）」というパッケージは、ニセコや白馬のような画一化された西洋風リゾートに飽きた欧米豪の富裕層・文化体験層（オーセンティシティを求める層）に深く刺さる可能性を秘めている [cite: 28]。
*   **通年化 (グリーンシーズンの活用)**: 夏期にはカート場「カートランド三川」として活用されている歴史があるが [cite: 4]、さらなる通年化として、三川観光きのこ園（BBQ施設）と連携した「ゲレンデキャンプ場」や、リフトを活用した「マウンテンバイク（MTB）ダウンヒルコース」の整備が考えられる。雪不足リスクをヘッジするための夏山リゾート化は急務である。
*   **DX (デジタルトランスフォーメーション)**: 従業員不足と現金決済の課題を同時に解決する一手として、DXの推進が不可欠である。Webでの事前リフト券購入システム（QRコード発券等）を導入し、チケット売り場の無人化・省力化を図る。また、ゲレ食のモバイルオーダー化により、限られた人員でピークタイムのオペレーションを回す体制を構築すべきである [cite: 35]。

## 結論：売上倍増に向けたアクションプラン

上記のリサーチと分析に基づき、三川温泉スキー場が売上を倍増させるための、現実的かつインパクトのあるアクションプランを提言する。

*   **短期アクション（1年以内で実行可能・即効性重視）**
    *   **通信・コスト課題をクリアしたキャッシュレス・事前決済のスモールスタート**: 完全現金主義の背景にある「決済手数料の負担」「通信インフラの不安」「端末導入コスト」といった障壁を越えるため、まずは初期費用が低くスマートフォン単体で処理可能なQRコード決済（PayPay等）から導入を開始する。通信リスクに対しては、安価な安定型モバイルルーターをチケット窓口周辺に配置することで対応する。「アソビュー！」等を通じたWeb事前購入システムも併せて構築し、「チケット窓口の混雑解消」と「販売スタッフの省人化」を即座に実現する [cite: 7, 14, 15, 16]。
    *   **「ゲレンデ × 秘湯」デジタルパッケージ券の創設**: 既に存在する「入浴料100円引き」というアナログな連携を一歩進め、三川温泉郷の旅館協同組合と連携し、「リフト1日券＋温泉入浴券＋温泉街での飲食クーポン」がセットになったデジタル周遊チケットを販売する。単なるスキー場としてではなく、阿賀町全体の「冬のレジャー体験」としてマーケティングを転換する [cite: 9, 11]。
    *   **SNSを駆使した「斜度38度」のバズマーケティング**: ファミリー層向けの広報に加えて、TikTokやInstagramのショート動画を活用し、ダイナミックコース（38度非圧雪）の過酷さとローカルゲレンデのギャップを押し出したプロモーションを行う。コアなスキーユーチューバー等を招聘し、「知る人ぞ知る激ヤバコース」としての認知を拡大させる。

*   **中長期アクション（1〜3年スパン・根本的な価値向上）**
    *   **外部資金を活用した「ICオートゲート」導入による抜本的省人化**: 慢性的な従業員不足をカバーするため、全リフトの稼働にこだわるのではなく、最も効率よく全コースにアクセスできる基幹リフトに最新のICオートゲートシステムを導入する。数千万円規模となるこの設備投資の財源として、スキー場単独の予算ではなく、国や県の「観光DX推進事業補助金」や、阿賀町の「デジタル田園都市国家構想交付金」などを戦略的に獲得する。または機器のリース契約によって初期費用を平準化し、監視員の省力化と顧客データ収集（利活用）を両立させる [cite: 7, 14, 35]。
    *   **スノーインディペンデント（脱・雪依存）な通年型リゾートへの脱皮**: 雪不足による営業日数減のリスクを根本から解決するため、夏季の「キャンプフィールド化」や、オフロードバギー、グラススキー、MTBコースの整備に投資する。地域おこし協力隊や外部の民間アウトドア企業を誘致し、グリーンシーズンの収益の柱を確立する。
