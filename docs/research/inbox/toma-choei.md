# 当麻町営スキー場 観光マーケティング・売上倍増戦略 深掘り調査レポート

本レポートは、完全無料の公営施設である「当麻町営スキー場」を対象とした売上倍増に向けたマーケティング戦略の深掘り調査レポートです。

**【エグゼクティブサマリ（結論）】**
*   **結論:** 施設単体での「リフト券販売」による従来の収益化ではなく、スキー場をハブとした「当麻町全体の経済波及効果（エリア消費単価）の最大化」および「高付加価値な付帯サービス（レンタル・飲食）による新規マネタイズ」が売上倍増の最適解となります。
*   **明確なターゲット:** 本格的なスキーヤー層ではなく、旭山動物園から流入する訪日外国人（インバウンド）と、雪遊び・ゲレンデデビューを果たす0次層のファミリー層に完全特化します。
*   **推奨アクションプラン:**
    1.  「雪遊び手ぶらセット」の高単価有料レンタル化
    2.  週末限定フードトラック誘致と最新スマート自販機（ど冷えもん等）の導入
    3.  スノーエスカレーターの導入（初心者エリアの有料化）による顧客体験の抜本的改善
    4.  周辺温泉・グルメと連携した「当麻ウィンターパス」のパッケージ販売

当麻町営スキー場（とうま山スキー場）は、旭川市中心部から車で約30分、旭山動物園からも約15分という絶好のアクセスを誇りながら、リフト代から駐車場まで完全無料で提供されている極めて特異なロコスキー場です[cite: 1, 2]。当施設は「町営」の名称が示す通り、本来は地域住民の健康増進やレクリエーション、そして地元の子どもたちのスキー・スノーボード教育（冬の体育）を目的として税金で整備された公共福祉施設としての歴史的背景を持ちます。そのため、利益追求を前提とせず無料で開放されてきました。本レポートでは、当施設が「無料の公営施設」であることを前提に据え、「売上倍増」という命題を「スキー場をハブとした当麻町全体の経済波及効果（エリア消費単価）の最大化」および「未開拓の付帯サービス（レンタル・飲食）による新規マネタイズ」と再定義して分析を行います。

調査結果から、当施設は本格的なスキーヤー層を近隣の競合施設に譲る一方で、「小さな子どもの雪遊び・ゲレンデデビュー」において圧倒的な優位性を持つことが明らかになりました[cite: 1]。一方で、飲食設備の欠如や、初心者には難易度が高いシュレップリフト（ロープリフト）の運用といった顧客体験上のボトルネックも存在します[cite: 1, 3]。本稿では、インバウンドの「雪遊びニーズ」や周辺の良質な温泉・グルメ資源とのシナジーを紐解き、具体的なアクションプランを提示します。

## 1. 基礎データと現状のポジショニング

当麻町の中央、当麻山の一角に位置する当麻町営スキー場は、地域住民の冬のレクリエーションや、子どもたちのスキー・スノーボードデビューを主眼に置いた零細規模のゲレンデです[cite: 1, 4]。

### 基本情報とゲレンデ・設備構成
スキー場を構成する基礎データは以下の通りです。
*   **所在地:** 〒078-1300 北海道上川郡当麻町市街6区[cite: 5, 6]
*   **連絡先:** TEL 0166-84-2698[cite: 5, 6]
*   **アクセス:** 旭川北ICから約15km。JR石北本線「当麻駅」からタクシーで約5分、または道北バス「ヘルシーシャトー」下車徒歩10分[cite: 5, 7]。
*   **規模とコース:** コース面積約8ha、最大滑走距離240m、最大斜度17度の緩斜面1コースのみの構成です[cite: 1, 7]。縦長というより四角くワイドな一枚バーンであり、初心者が恐怖心を抱きにくい設計となっています[cite: 1]。
*   **リフト設備:** 腰当付きのシュレップリフト（ロープリフト）が1基のみ稼働しています[cite: 1]。ゴンドラやチェアリフトは存在しません。
*   **その他設備:** ゲレンデとは完全にネットで仕切られた「ボブスレー（ソリ）専用コース」や、当麻山を一周する3kmの歩くスキー（クロスカントリースキー）コースが完備されています[cite: 1, 7]。
*   **料金体系:** リフト利用料、ボブスレーコース利用料、および200台収容の駐車場料金のすべてが「完全無料」です[cite: 7]。
*   **付帯施設:** 無料の休憩所（ロッジ）とトイレのみ。ロッジにはフリーWi-FiとAEDが完備されています[cite: 1, 8]。

### ターゲット層と市場の立ち位置
現在の主要ターゲットは、旭川市および当麻町近隣に居住するファミリー層、特に「スキー・スノーボードを初めて体験する小さな子ども」です[cite: 1, 9]。本格的な滑走を楽しむ層は近隣の中〜大規模スキー場へ流れるため、当施設は「雪遊びの入り口」としての役割（受け皿）に完全に特化するポジショニングを確立しています[cite: 1, 10]。

## 2. 強みと弱み (内部環境)

無料のロコスキー場という特性上、強みと弱みは表裏一体の関係にあります。内部環境の分析から、集客の武器となる要素と、顧客満足度を削ぐボトルネックを抽出します。

### 強み (Strengths)
*   **圧倒的なコストパフォーマンス:** リフト代・駐車場代が完全無料であることは、家族連れにとって最強のフックです[cite: 7]。
*   **初心者に特化したゲレンデ形状:** 幅が広く、斜度にキツい部分がない素直な緩斜面は、ゲレンデデビューに最適です[cite: 1]。
*   **安全な雪遊び環境:** スキー滑走エリアとソリ遊び（ボブスレー）エリアがネットで明確に分離されており、衝突の危険性が低減されています[cite: 1]。
*   **ホスピタリティ:** ロープリフト乗り場には係員が常駐し、転倒した子どもの救助やストックの回収など、安全管理とサポートに奔走しており、安心感につながっています[cite: 1, 3]。

### 弱み (Weaknesses)
*   **リフトの難易度と運行効率:** 唯一の機動力である「シュレップリフト」は、一般的なスキー場にあるような「座って移動するチェアリフト」とは構造が根本的に異なります。利用者は雪面上に立ったまま専用の搬器（バーやロープ）に跨がる、あるいは腰にあてがい、ワイヤーに引っ張られながら雪面を滑り上がる仕組み（滑走式リフト）です[cite: 11, 12]。自らの足と板でバランスを取り続ける必要があるため、乗り慣れない子どもや初心者には難易度が高く、転倒によるリフトの緊急停止が頻発します。これにより順番待ちの時間が長引き、顧客のストレス要因となっています[cite: 1]。
*   **飲食設備・売店の欠如:** 施設内には食事を提供する場所や売店が一切ありません。長時間の滞在には事前の食料調達が必須であり、スキー場内での消費（マネタイズ）機会を完全に損失しています[cite: 1, 3]。
*   **インフラ管理の甘さ:** 駐車場はゲレンデとフラットに繋がっておりアクセスは良好ですが、降雪後の除雪体制が不十分な場合があり、車から降りた途端に足が深く埋まるといった不満が見受けられます[cite: 1]。

旭川市中心部からのアクセスに優れ、3基のリフトと多様なコース長・ゲレンデの広さを誇る道北圏の主要公営スキー場です[cite: 9, 15]。
*   **当麻の優位性:** 圧倒的な低価格（無料）と、ソリ遊び専用エリアの手軽さ。
*   **当麻の劣後要因:** コースの多様性、滑走距離、リフトの快適性（チェアリフトの有無）、付帯施設の充実度で大きく劣ります[cite: 9]。

*   **当麻の優位性:** 繰り返しになりますが「完全無料」という点に尽きます。また、完全な初心者のみが滞在するため、上級者と交錯する恐怖感がありません[cite: 1]。
*   **当麻の劣後要因:** コースのバリエーションと施設インフラの質。

**競合に対する結論:** 
当麻町営スキー場は、スキーヤー・スノーボーダーをターゲットとした「滑走体験」で競合と戦うべきではありません。「雪に触れる」「ソリで遊ぶ」「とりあえず板を履いてみる」というプレ・スキーヤー層（0次層）の獲得において、競合を排除できる独自のポジションを持っています。

## 4. ターゲット層のインサイト（口コミ分析）

Web上の口コミや訪問記から、実際の利用者が抱いているインサイトを分析します。

### ポジティブな傾向
顧客は「無料であること」に最大限の価値を感じており、設備が最低限であっても納得感を持っています。「ロープトーがなんと無料！」「無料なのが良い」といった声が目立ちます[cite: 3]。また、「係員さんがしっかり安全管理してくれている様子で、安心感あり」と、ハード面の不足をスタッフの属人的な努力でカバーしている点が高く評価されています[cite: 1, 3]。

### ネガティブな傾向と隠れたニーズ
「施設には売店などの施設がないため、食事の用意などはお忘れなく！」と口コミで注意喚起されているように、飲食の欠如は明確なペインポイントです[cite: 3]。
また、「シュレップリフトを使い慣れない子供が多く、度々リフトがストップする」という点から、**「もっと簡単に斜面を登れる設備（ムービングベルトなど）があれば、さらに快適に雪遊びができるのに」**という隠れたニーズが浮き彫りになっています[cite: 1]。

## 5. 周辺観光資源とのシナジー（エリア回遊性）

当麻町営スキー場の最大のポテンシャルは、徒歩・車で数分圏内に良質な観光資源とグルメが密集している点にあります。スキー場をハブとしてこれらを回遊させることで、町全体の消費単価を劇的に向上させることが可能です。

### 当麻健康福祉施設 ヘルシーシャトー（温泉・サウナ）
スキー場から約2.3km（車で数分、徒歩圏内）に位置する温浴施設です。長万部二股温泉のラジウム鉱石を利用した人工温泉で、サウナや気泡浴も完備しています[cite: 16, 17, 18]。入浴料は大人600〜700円と安価であり、「スキー場でたくさん遊んだ後に、身体を温めるのに最適」と口コミでも定評があります[cite: 2, 19]。

### 周辺の強力なご当地グルメ
スキー場周辺には、わざわざ遠方から訪れる価値のある飲食店が集積しており、それぞれが明確な価格帯と魅力を持っています。
*   **ピザハウス ココペリ:** スキー場の向かいの森の中にあるログハウスレストラン。本格的な石窯ピザ（フォカッチャやカルツォーネ）が楽しめ、常に満席の人気店です[cite: 20, 21]。予算は1,000円〜1,999円程度で、質の高いメニューが提供されています[cite: 22]。
*   **MOONLOID Cafe:** アパレルショップに併設された洗練されたカフェ。ピタパンサンドや副菜・スープがセットになったボリューム満点の「ムーンロイドランチ」が1,800円（税込）で提供されており、客単価を高められる良質なスポットです[cite: 21, 23, 24]。
*   **北の厨房:** ヘルシーシャトー内にあり、普通盛りでも他店の大盛りサイズに匹敵する「チキンカツカレー」や、定番の「とんかつ定食（930円）」、「手作りチャーシューの正油ラーメン（700円）」など、600円〜1000円以下の価格帯でスキー後の空腹を満たすのに最適な食堂です[cite: 25, 26]。

### エリア回遊ルート構築案
**「雪遊びデビュー ＆ 当麻満喫 黄金ルート」**
午前中に当麻町営スキー場でソリ遊びやスキーデビューを実施。昼食はスキー場向かいの「ココペリ」で絶品ピザを堪能（またはキッチンカーをスキー場に誘致）。午後は「ヘルシーシャトー」で冷えた身体をサウナとラジウム温泉で温め、施設内の「北の厨房」で夕食を取るか、当麻駅周辺の「麺や和泉」でラーメンを食べて帰路につくルートです[cite: 19, 25, 27]。

## 6. 最新トレンドの適応可能性

市場の最新トレンドを踏まえ、当施設が今後取り入れるべき新しい方向性を考察します。

### インバウンド：訪日外国人客の取り込み
旭川市に隣接し、インバウンドのキラーコンテンツである「旭山動物園」からわずか10km（車で約15分）という立地は絶大です[cite: 2]。雪に触れたことのない東南アジア等のファミリー層にとって、本格的なスキー場は敷居が高すぎますが、「無料でソリ遊びができる広い緩斜面」は理想的な環境です。「旭山動物園の帰りに立ち寄る、無料のSnow Play Park」としてのプロモーション余地は無限大にあります。

### 通年化：グリーンシーズンの活用
当麻山の麓には「とうまスポーツランドキャンプ場」や木育施設「くるみなの木遊館」、フィールドアスレチックなどが集積しています[cite: 28, 29, 30]。グリーンシーズンはキャンプ客のレクリエーション拠点として、傾斜を利用したマウンテンバイク（MTB）のグラスコースや、芝生ソリのゲレンデとして通年活用するポテンシャルを秘めています。

### 新規軸：ワーケーションとサステナビリティ
ロッジにはすでにフリーWi-Fiが完備されています[cite: 8]。これを活かし、親はロッジ内の窓際でテレワーク（ワーケーション）を行いながら、子どもたちは目の前の安全なゲレンデでソリ遊びを満喫するという、現代のライフスタイルに合わせた新しいスキー場の利用形態を提案できます。

## 結論：売上倍増に向けたアクションプラン

当麻町営スキー場は施設自体が完全無料であるため、「売上倍増」は「スキー場を起点とした町内での消費額倍増」および「新たな付加価値サービスへの課金」によって達成します。

### 短期アクション（1年以内で実行可能・即効性重視）

*   **「雪遊び手ぶらセット」の有料レンタル化（インバウンド・動物園客対応）**
    スキー板だけでなく、高品質なスノーウェア、長靴、手袋、プレミアムなソリ（スノーストライダーや大型ボブスレー）の「手ぶら雪遊びレンタルパッケージ」をロッジで有料提供します。旭山動物園帰りの「急に雪遊びがしたくなった」観光客を確実にマネタイズします。
*   **週末限定のフードトラック（キッチンカー）誘致とスマート自販機の導入**
    飲食設備ゼロの弱みを逆手に取り、週末や休日に旭川・当麻エリアのキッチンカーを駐車場に誘致し、出店料を徴収（または売上シェア）します。平日対策としては、温かい食事（ホットスナックやカップ麺）や本格的な冷凍食品を提供する最新の無人スマート自販機（サンデン・リテールシステム社の「ど冷えもん」など）をロッジ内に設置し、人件費をかけずに安定した売上を創出します。
*   **当麻周遊・電子クーポンの発行**
    スキー場のロッジ内にQRコードを掲示し、「スキー場来場者限定：ヘルシーシャトー入浴割引＆ココペリ・MOONLOID等でのワンドリンクサービス」のデジタルクーポンを発行。スキー場をハブとした町内消費をトラッキングしつつ促進します。

### 中長期アクション（1〜3年スパン・根本的な価値向上）

*   **シュレップリフトから「スノーエスカレーター（ムービングベルト）」への段階的移行**
    顧客の最大のペインポイントである「リフトに乗れない」問題を解決するため、初心者やソリ遊び客が安全に登れるムービングベルトを導入します。具体的なロジスティクスとして、青森県の「十和田湖温泉スキー場」では、スノーパーク向けの長さ約40mのスノーエスカレーターを、本体・電気工事費込みで約1,620万円の予算規模で新設し、劇的な集客増につなげた実例があります[cite: 31]。これを「初心者用スノーパークエリア」として区画し、このベルト利用のみを「1日500円」程度の有料（町民は無料・割引）とすることで、インバウンドや観光客から新たな収益の柱を確立します。
*   **「当麻ウィンターパス」の造成によるパッケージ販売**
    「ムービングベルト利用券 ＋ ヘルシーシャトー入浴券 ＋ 周辺飲食店で使える1,000円分食事券」をセットにした『当麻ウィンターパス』を造成し、旭川市内のホテルやオンライン（OTA）で販売します。OTA（Online Travel Agency）とは、インターネット上だけで取引を行う旅行会社のことで、例えるなら「旅行予約のためのデジタルスーパーマーケット」です。じゃらんやBooking.com等のOTAを活用することで訪日客への販路を広げ、単なる無料スキー場から「1日遊べる滞在型スノーリゾート」へと脱皮させ、町全体の経済波及効果を倍増させます。

### 補足：公営施設マネタイズに向けた行政的・制度的ハードルの突破
