# Executive Summary

本レポートは、福井県鯖江市に位置する「莇生田（あぞうだ）スキー場」の現状分析および売上倍増・再興に向けた戦略的インサイトをまとめたものである。
*   **基礎データと現状**: 鯖江市の住宅街から至近に位置する極小規模ゲレンデ（全2コース、最大滑走距離250m）。現在は圧雪車の故障と近年の雪不足により実質的な休業（閉鎖）状態にある。
*   **内部環境の強みと弱み**: 地域住民の深い愛着と圧倒的なアクセスの良さが最大の武器。一方で、インフラ不全と滑走体験の欠如、気候変動への脆弱性が致命的な弱点である。
*   **競合に対する決定的な優位性**: 「今庄365スキー場」や「スキージャム勝山」等の周辺リゾートと比較し、移動時間がほぼゼロであり、かつリフト代300円という「手軽な雪遊び場」としてのニッチなポジションでは他を圧倒している。
*   **隠れたニーズ**: 本格的なスキー需要よりも、「近場で安全に子供を遊ばせたい」「雪に依存しない新しい遊び場（ドッグランやMTB等）が欲しい」という、空間利活用への潜在ニーズが非常に高い。
*   **ボトムライン・アクションプラン**: 単なるスキー場としての単独再建を諦め、クラウドファンディングを通じた資金調達と、無人スマートロックを活用した「完全無人化・通年型アウトドアパーク」へのピボット（用途転換）を推奨する。

# 莇生田スキー場 売上倍増・再興に向けた戦略調査レポート

**JAPOW ID: 365**

本レポートは、福井県鯖江市に位置する「莇生田（あぞうだ）スキー場」の現状分析および売上倍増に向けた戦略的インサイトを提供するものです。当スキー場は現在、圧雪車の故障および近年の雪不足により「休業（実質的な閉鎖状態）」に陥っています [cite: 1, 2]。したがって、本レポートにおける「売上倍増」とは、実質ゼロとなっている現在の収益構造を抜本的に見直し、グリーンシーズンの活用や用途転換（ピボット）を通じて、地域密着型の新たな収益基盤をゼロから創出・最大化するためのアクションプランを提示します。施設は小規模なローカルゲレンデですが、地域住民からの愛着は深く、新たなアウトドア拠点として再生するポテンシャルを秘めています。

## 1. 基礎データと現状のポジショニング

莇生田スキー場は、鯖江市東部の河和田地区（莇生田町）に位置する極めて小規模な市営ゲレンデです [cite: 2, 3]。現在は設備の故障により営業を休止しています [cite: 1, 2]。

*   **基本情報とアクセス**
    *   **住所**: 〒916-1224 福井県鯖江市莇生田町 [cite: 3, 4]
    *   **連絡先**: 0778-53-2260（鯖江市教育委員会スポーツ課） [cite: 1, 4]
    *   **Google MAP**: [https://www.google.com/maps/search/?api=1&query=福井県鯖江市莇生田町+莇生田スキー場](https://www.google.com/maps/search/?api=1&query=福井県鯖江市莇生田町+莇生田スキー場)
*   **規模とコース構成**
    *   **ゲレンデ規模**: 全体で約3ヘクタールという零細規模 [cite: 5]。
    *   **標高・斜度**: ベース標高70m、トップ標高135m（標高差わずか65m）。最大滑走距離は約250m（体感150m程度）、最大斜度は15度です [cite: 2, 5]。
    *   **コース割合**: コース数は全2本。初級者向けが80%、中級者向けが20%、上級者向けは0%の完全な緩斜面構成となっています [cite: 2, 5]。
*   **主要設備（リフト・ゴンドラ等）**
    *   ゴンドラやペアリフト等の大型設備は一切存在しません [cite: 2]。
    *   過去には簡易的なロープトゥ（Jバーリフト）が2基ありましたが、近年は下部に1基（ワイヤーのみの状態で実質非稼働）が残存するのみとなっており、上部コースへは自らの足で登る（ハイクアップ）必要がありました [cite: 2, 5]。
    *   その他、30台収容の駐車場と、無料の休憩所・トイレを備えた小規模なセンターハウスが存在します [cite: 1, 5]。
*   **料金体系**
    *   営業時のリフト利用料は「3時間で300円」という破格の設定でした（入場料無料） [cite: 1, 2]。
*   **ポジショニング**
    *   本格的なスキー・スノーボードリゾートではなく、ゲレンデの真下まで住宅地が迫る「近隣住民のソリ遊び・雪遊び場」としての立ち位置です [cite: 5, 6]。

## 2. 強みと弱み (内部環境)

施設の内部環境を分析すると、スキー場としてのハード面の弱点が目立つ一方で、地域コミュニティにおけるソフト面の強みが浮き彫りになります。

*   **強み (集客の武器)**
    *   **圧倒的なアクセスの容易さと生活密着性**: 山奥に車を走らせる必要がなく、鯖江市の住宅街から極めて近い立地です [cite: 5, 6]。アップダウンの少ない道のりで到達できる利便性があります [cite: 5]。
    *   **地域住民のノスタルジーと愛着**: オープンから40年以上の歴史があり、雪が降れば連日ソリ遊びの家族連れで賑わうなど、市民の原風景として深く根付いています [cite: 7]。
*   **弱み (課題・ボトルネック)**
    *   **致命的なインフラ不全**: 圧雪車の故障によりゲレンデ整備ができず、施設全体が休業（閉鎖）に追い込まれています [cite: 1]。また、リフト（Jバー）も実質的に機能していません [cite: 2, 5]。
    *   **気候変動への脆弱性**: ベース標高が70mと非常に低く、人工降雪機もないため、近年の暖冬・雪不足の影響をダイレクトに受けます [cite: 2, 5, 6]。
    *   **滑走体験の欠如**: 緩斜面かつ滑走距離が短いため、中級者以上のスキーヤー・スノーボーダーを満足させる要素が皆無です [cite: 2, 5]。

    *   **九頭竜スキー場**（大野市）：同じく休止から再開を果たしたローカルゲレンデ。緩斜面から最大41度の急斜面まで全7コースを有します [cite: 8, 9, 11]。
    *   **スキージャム勝山**（勝山市）：西日本最大級の規模を誇るリゾート。全16コース、最長滑走距離5,800mを誇ります [cite: 8, 12]。

## 4. ターゲット層のインサイト（口コミ分析）

過去の訪問者や地元住民の声を分析すると、スキー場としての機能に対する諦めと、空間そのものに対する強い期待が混在しています。

*   **顧客の実際の声（ポジティブ・ネガティブ）**
    *   **ポジティブ**: 雪が積もった際の喜びの声は大きく、「再びにぎわいが生まれうれしい」といった地域住民の歓迎ムードが見られます [cite: 7]。
    *   **ネガティブ（実態）**: 「滑走する人はほとんどいない」「ソリ遊びの家族がいるだけ」「リフトのバーがなく運行していない」「コースに落ち葉や泥が上がっている」など、スキー場としての体をなしていないことに対するリアルな指摘が多数存在します [cite: 5]。
*   **顕在化していない隠れたニーズ**
    *   「スキーを滑りたい」というニーズよりも、「近場で子供を安全に遊ばせたい」「自然の中で過ごしたい」というニーズが本質です。現在、地元の市議会議員や有志の間で「ドッグラン」「ドローンの練習場」「マウンテンバイク場」など、雪に依存しない新しい遊び場としての利活用案が活発に議論され始めており、空間の再利用に対する強い潜在ニーズが存在します [cite: 6]。

## 5. 周辺観光資源とのシナジー（エリア回遊性）

鯖江市は「めがね」や「越前漆器」といった世界的にも有名なものづくりの街であり、独自の食文化や温泉も充実しています。これらを結びつける回遊ルートの構築が単価向上の鍵となります。

*   **近隣の有望な観光資源**
    *   **ご当地グルメ**: ゲレンデのある河和田地区に伝わる伝統薬味「山うに（柚子と唐辛子の薬味）」を使った「ほやっ停」の山うにたこ焼き [cite: 15, 16, 17] や、福井名物ソースカツ丼の「ヨーロッパ軒 神明分店」 [cite: 18, 19]、地元で愛される「御園飯店」の餃子 [cite: 20, 21, 22] などが強力なコンテンツです。
    *   **温泉・宿泊**: 美人の湯と長寿の湯の混合泉を楽しめる「かわだ温泉 ラポーゼかわだ」 [cite: 23, 24] や、「三六温泉 神明苑」 [cite: 25, 26]、サウナが充実したスーパー銭湯「越のゆ 鯖江店」 [cite: 27, 28] が至近にあります。
    *   **観光スポット**: 「めがねミュージアム」での眼鏡作り体験、「うるしの里会館」での越前漆器の絵付け体験、レッサーパンダで有名な「西山動物園」など、ファミリー層向けの文化・レジャー施設が豊富です [cite: 29, 30, 31]。
*   **回遊ルート構築案（ものづくり×ローカルアウトドア）**
    *   **午前**: 莇生田スキー場跡地でのアクティビティ（冬場はソリ遊び、グリーンシーズンはドッグランやドローン体験）。
    *   **昼食**: 「ほやっ停」で山うにたこ焼きをテイクアウト、またはヨーロッパ軒でカツ丼 [cite: 17, 19]。
    *   **午後**: めがねミュージアムやうるしの里会館で、家族揃って伝統工芸の制作体験 [cite: 30]。
    *   **夕方**: ラポーゼかわだ（温泉）で冷えた・疲れた体を癒やし、施設内でBBQや夕食をとる [cite: 23, 24]。
    *   このルートにより、スキー場単体では300円だった顧客単価を、飲食・体験・温泉を含めたエリア全体で1人当たり5,000円〜10,000円規模へ引き上げることが可能です。

## 6. 最新トレンドの適応可能性

休業中の小規模ゲレンデを再生するためには、最新トレンドの導入による大胆なピボット（路線変更）が不可欠です。

*   **インバウンド (訪日外国人客)**
    *   現状のスキー設備でインバウンドを惹きつけることは不可能です。しかし、「Deep Japan（日本の原風景と伝統文化）」を探求するニッチな層に対し、「越前漆器の里（河和田地区）の雪景色でのソリ遊びと、漆器工房見学」をパッケージ化することで、スノーリゾートとは異なる文化的体験として提供する余地があります [cite: 6, 30]。
*   **通年化 (グリーンシーズンの活用)**
    *   **最重要課題**です。地元の有志が提案している通り、急斜面を利用した「マウンテンバイク（MTB）コース」、広大な芝生を利用した「大型ドッグラン」、障害物のない空域を活かした「ドローン練習場」への転用は極めて現実的かつ有望です [cite: 6]。また、近隣で同じく閉鎖が予定されている「尾花キャンプ場」と連携し、一帯をアウトドアパークとして再定義することが求められます [cite: 4, 6]。
*   **新規軸 (サステナビリティ・DX)**
    *   **クラウドファンディングと地域参加**: 故障した圧雪車の修理費用や、ドッグラン整備費用をクラウドファンディングで募り、出資者を「施設の共同オーナー」として巻き込むサステナブルな運営モデルが有効です。
    *   **無人化・DX**: 長野県の白馬岩岳マウンテンリゾートにおける非接触型スマートエントリーの成功事例や、都市部の遊休地で普及している無人ドッグラン施設のように、グリーンシーズンの運用においては最新技術が不可欠です。施設エントランスに「Akerun」等の無人スマートロックを導入し、「STORES 予約」等のQRコード決済プラットフォームと連携させた「完全無人エントリーシステム」を構築することで、人件費（OPEX：Operating Expense/運用コスト）を極限まで抑えた運営が必須となります。

## 結論：売上倍増に向けたアクションプラン

実質的に売上がゼロ（休業中）となっている現状から、新たな収益基盤を創出し、過去最高益へと倍増させるための現実的なステップを提案します。「スキー場」としての復権ではなく、「地域密着型の通年アウトドア拠点」へのトランスフォーメーションが必須です。

*   **短期アクション（1年以内で実行可能・即効性重視）**
    1.  **「グリーンシーズン特化型パーク」への暫定オープン手続きと実証実験**:
        *   **ステップ1**: 鯖江市役所（教育委員会スポーツ課または都市計画課等）に対し、「行政財産使用許可申請」または「都市公園法に基づく占用許可申請」を提出し、スキー場跡地の目的外使用（ドッグランやドローン練習場）の認可を取得する。
        *   **ステップ2**: 決済プラットフォーム（例: STORES 予約、Square等）と、無人スマートロック（例: Akerun）を連携させたゲートシステムを導入し、完全無人での課金・入場フローを構築する。
        *   **ステップ3**: コースの草刈り等の最低限の整備を行い、週末限定でのテストオープン（場所貸し）を実施する。
    2.  **「莇生田ゲレンデ・リバイバル」クラウドファンディングの実装**:
        *   **ステップ1**: 大手クラウドファンディングサイト（CAMPFIRE または Makuake）にてプロジェクトページを開設。「故障した圧雪車の修理」または「通年型パーク化のための初期投資」を明確なゴールに設定する。
        *   **ステップ2**: 資金提供者への返礼品（リターン）として、「ゲレンデのネーミングライツ（命名権）」、「河和田の越前漆器」、「地元グルメの山うに」などを設定し、地元企業とのアライアンスを組む。
        *   **ステップ3**: プレスリリースを配信し、SNS広告等を通じて鯖江市民の郷土愛に直接訴えかけ、ノーリスクで資金調達を実行する。

*   **中長期アクション（1〜3年スパン・根本的な価値向上）**
    1.  **「ものづくり×スノープレイ」のパッケージ観光事業化と販路開拓**:
        *   **ステップ1**: 地元の「めがねミュージアム」「うるしの里会館」「ラポーゼかわだ」などの主要施設とプロモーション協議を行い、「鯖江伝統工芸体験 ＋ 莇生田でのプライベート雪遊び ＋ 温泉」をセットにした冬の高単価パッケージツアーを造成する。
        *   **ステップ2**: OTA（じゃらん、楽天トラベル、Klookなどのオンライン旅行予約サイト）に「鯖江クラフト＆アウトドア体験」として登録し、販路を拡大する。
        *   **ステップ3**: 小規模なインバウンド層や高単価ファミリー層に向けた多言語対応のLP（ランディングページ）を展開する。
    2.  **尾花キャンプ場と統合した「河和田アウトドア・エコヴィレッジ」構想の推進**:
        *   **ステップ1**: 閉鎖が検討されている近隣の「尾花キャンプ場」と「莇生田スキー場」を一体のエリアと見なし、鯖江市に対してPFI（民間資金等活用事業）や指定管理者制度の提案を行う。
        *   **ステップ2**: 大手アウトドアブランド（スノーピークやモンベル等）に協賛やネーミングライツを打診し、民間のノウハウを導入して行政負担を軽減させる。
        *   **ステップ3**: 夏はキャンプとMTB、冬はソリ遊びという「通年リゾート化」のマスタープランを策定し、地域産業と連動したエコヴィレッジを本格稼働させる。
