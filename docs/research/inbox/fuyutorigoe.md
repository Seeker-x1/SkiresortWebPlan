# 売上倍増戦略レポート：冬鳥越スキーガーデン

## Executive Summary (エグゼクティブ・サマリー)
本レポートは、新潟県加茂市に位置する「冬鳥越スキーガーデン」の売上倍増に向けた包括的な戦略を提示する。
- **現状と強み:** 県央エリア唯一のスキー場として、1日1,500円という圧倒的な低価格と、大正・昭和初期の木造電車が静態保存される独自景観を有し、子育て世帯の「ソリ遊び・雪遊びデビュー」において確固たる地位を築いている。
- **課題とボトルネック:** 用具レンタルが一切ないこと、そして通常のリフトが存在せず牽引式の「Tバーリフト」のみであるという物理的制約が、新規顧客やスノーボーダーの参入障壁となっている。また、単価向上の仕組み（ゲレンデ食や物販）も欠如している。
- **競合とのポジショニング:** 三川温泉スキー場などの本格的な周辺リゾートとは真っ向勝負を避け、「完全なファミリー特化」「エリア回遊の起点」としてのポジションを強化することが戦略の要となる。
- **財務的・戦略的提言:** 短期的には、近隣の「加茂七谷温泉 美人の湯」や市街地グルメと連携したセット販売（デジタルチケット化）や移動式ポップアップ・レンタルを通じ、顧客単価を現在の1,500円から3,000円〜4,000円台へ引き上げる。中長期的には、通年利用可能なプレミアム・デイキャンプの整備や最新の無人決済システムを導入し、人員を増やさずに収益性を劇的に改善するアクションを実行する。

## 1. 基礎データと現状のポジショニング

冬鳥越スキーガーデンは、加茂市が運営する市営の小規模ゲレンデである [cite: 7]。かつては通常のリフトを備えた歴史あるスキー場であったが、少雪や競合の影響による一時閉鎖を経て、ファミリーやそり遊び向けに特化して再開された経緯を持つ [cite: 7]。

**基本情報と主要設備**
ゲレンデは初級コースを中心に構成されており、初心者や子供連れが安心して利用できる設計となっている [cite: 5, 6]。公式発表では「全5コース」とされているが、個別のコース名称や各々の距離・斜度といった詳細な公式スペックは公開されていないため、判明している部分データに基づく構成は以下の通りである。
- **規模・コース構成:** 全体として初級60％、中級30％、上級10％の割合であり、最長滑走距離は220m、最大傾斜は20度である [cite: 6, 8]。
  - **メインゲレンデ:** 緩斜面中心。滑走距離約50m〜220m。初級者の基礎練習に適している。
  - **サイドゲレンデ:** メインゲレンデの左側に位置し、やや斜度が増す。滑走距離約50mの中級者向け。
  - **ソリ専用コース:** メインコースから完全に分離された安全なキッズ向けエリア [cite: 6, 9]。
- **リフト設備:** 通常のイス型リフトやゴンドラは存在せず、滑走者をワイヤーで引っ張り上げる「Tバーリフト（シュレップリフト）」が3基（資料によっては2基稼働、長さは約50m〜220mの範囲）設置されている [cite: 5, 6, 9, 10]。
  - *（※Tバーリフトとは：空中に吊るされる通常のイス型リフトとは異なり、T字型のバーを腰や股に当て、雪面をスキーやボードに乗ったままの状態で引きずられながら登る牽引式の仕組みである。特に両足を固定して横向きに乗るスノーボーダーや、エッジ操作に不慣れな初心者にとっては、バランスを崩しやすく転倒のリスクと物理的な疲労が伴うため、大きなハードルとなっている。）*
- **リフト稼働日と営業状況:** Tバーリフトの稼働日は「火曜、木曜、土曜、日曜、祝日」に限定されている [cite: 6, 11]。しかし、リフトが動かない月曜、水曜、金曜であっても、8:30〜17:00の時間帯でスキー場自体は開園しており、無料でソリ遊びや雪遊びを楽しんだり、ロッジを利用することが可能である [cite: 6, 11]。
- **主要設備:** ロッジ（無料休憩所）には、1階にコインロッカーと自動販売機（ホットスナック等）、2階に更衣室とゲレンデを一望できる大広間が備わっている [cite: 6, 7, 8]。用具のレンタルは行っていないため、来場者は事前に「桑田屋スポーツ」や「小林スポーツ」などの地元のスポーツ用品店で自前調達して持ち込む必要がある [cite: 3, 12]。
- **文化財の展示:** ゲレンデ内に、新潟県内最古の木造電車「モハ1（大正12年製造）」、半鋼製電車「モハ61（昭和15年製造）」、電気機関車「ED1」が静態保存されており、自由に見学が可能である [cite: 6, 13, 14]。
- **アクセスと料金:** 北陸自動車道「三条燕IC」から車で約50分、JR信越本線「加茂駅」から車で約15〜20分（予約制のりあいタクシーも利用可能）[cite: 6, 10, 15]。リフト1日券は大人1,500円、小人750円と極めて安価に設定されている [cite: 5, 6, 10]。

**市場におけるターゲット層と立ち位置**
現在の主なターゲット層は、近隣地域（県央エリア）に居住する未就学児〜小学生の子供を持つ子育て世帯である。県央エリア唯一のスキー場という独自性を持ち、「本格的な滑走」よりも「休日の雪遊びデビュー」「格安でのソリ遊び」を目的としたライトユーザー層に対して、確固たる立ち位置を築いている [cite: 5]。

## 2. 強みと弱み (内部環境)

施設の内部環境を分析すると、一般的なスキーリゾートとは異なる、ローカル・ガーデン型ならではの明確な武器と課題が浮き彫りになる。

**強み:** 
- **圧倒的なコストパフォーマンスと安全性:** 1日券1,500円という価格設定は、リフト券が高騰する現代のスキー市場において強力な武器である [cite: 16]。また、斜面が緩やかでソリ専用エリアが分離されているため、幼児の雪遊びデビューにおいて安全性が極めて高い [cite: 5, 10]。
- **「レトロ電車」による唯一無二の景観:** ゲレンデ内に大正・昭和初期の木造電車が置かれている景観は珍しく、ノスタルジックな雰囲気を醸し出している [cite: 6, 13, 14]。これはSNSでの写真映え（フォトジェニック）要素として高いポテンシャルを秘めている。
- **グリーンシーズンの実績:** 冬季だけでなく、約150種・約2,600本が咲き誇るバラ園や、直径20mの花時計があり、春から秋にかけてもピクニック広場やアマチュアカメラマンの撮影スポットとして一定の集客力を持っている [cite: 7, 13, 17]。

**弱み:**
- **ハード面での大きな制約（リフトとレンタル）:** スキー・スノーボードの用具レンタルを一切行っていないため [cite: 3]、市外からの来訪者やインバウンド客が「道中でレンタルして手ぶらで寄る」ことが困難であり、衝動的な来場を強く阻害している。また、Tバーリフトのみであることは、技術的・体力的なハードルとなっている [cite: 6, 7]。
- **コース規模の限界と天候依存:** 最長滑走距離220m・最大傾斜20度というスペックは、中・上級者にとっては物足りない規模である [cite: 6]。さらに標高が低いため、昨今の少雪・暖冬傾向の直撃を受けやすく、積雪状況によって閉園を余儀なくされる日が多い [cite: 4, 7]。
- **収益性の低さ:** 驚異的な低価格は集客の強みである反面、顧客単価が極端に低くなる原因でもある。ゲレンデ内での飲食提供が自動販売機等に留まっており [cite: 6, 7, 8]、滞在中の付加価値による収益化（マネタイズ）の仕組みが欠如している。

**優位性と劣後要因の分析**
冬鳥越スキーガーデンの最大の優位性は、「県央エリアからの物理的な近さ」と「1,500円という価格による完全な雪遊び特化」である [cite: 5, 6]。競合が「本格的に滑るスキーヤー・スノーボーダー」をメインターゲットにしているのに対し、冬鳥越は「ソリ遊び」に主眼を置いているため、そもそも本格的なスキー場との競合を回避できている。また、古い電車の展示 [cite: 13] や広々としたロッジの無料開放 [cite: 7] は、小さな子供を連れて休憩を挟みながら遊ぶ用途において利便性が高い。
一方で、設備投資においては他施設に大きく劣後している。競合施設が安定したリフト輸送力、レンタルショップ、充実したゲレンデ食を提供しているのに対し、冬鳥越はこれら全てが欠落している [cite: 3, 6, 8]。

## 4. ターゲット層のインサイト（口コミ分析）

ウェブ上のレビューや口コミから、利用者の顕在化している評価と隠れたニーズを分析する。

**ポジティブな傾向**
冬季の口コミでは「小さい子供でも安心して遊べる」「アットホームな雰囲気」「お財布に優しい」といった、ファミリー層からの絶大な支持が見受けられる [cite: 5, 6, 13]。また、グリーンシーズンにおける口コミも多く、「バラがとても美しい」「犬の散歩がてらよく行く」「人が少なくておすすめ」と、静かで自然豊かな環境が高く評価されている [cite: 28, 29]。

**ネガティブな傾向と隠れたニーズ**
一方で、「Tバーリフトのみであることの不便さ」がスキーヤー・スノーボーダーの成長を阻害している面がある。ソリ遊びからステップアップしてスキーを始めようとする子供にとって、座って乗るリフトがないことは足腰への負担と技術的ハードルを生んでいる。
また、隠れた不満点として「食の充実度の低さ」が挙げられる。ロッジの2階大広間で持ち込んだお弁当を食べるアットホームさは良いが [cite: 7]、現地で温かく質の高い食事を購入できないことは、滞在時間の短縮（午前中だけ遊んで帰るなど）を招いている。

## 5. 周辺観光資源とのシナジー（エリア回遊性）

冬鳥越スキーガーデン単体では顧客単価に限界があるため、近隣の魅力的な観光資源と連携した「エリア回遊ルート」の構築が必須である。

**近隣の有望な観光資源**
- **加茂七谷温泉 美人の湯:** スキー場から車で近距離にある日帰り温泉施設。名峰・粟ヶ岳を望む露天風呂やハニカムサウナを備え、併設の「ななたに食堂」では新潟名物タレカツ丼や地元の食材を使った「ななたに御膳」、鴨だしが特徴の「愛KAMOらーめん」などのグルメを提供している [cite: 30, 31, 32, 33]。
- **加茂山公園（加茂市街地）:** 加茂駅から徒歩圏内にある都市公園。広大な敷地に「雪椿園」があり、シマリスと直接触れ合える「加茂山リス園」が併設されている（ただしリス園は冬季冬眠のため休園）[cite: 34, 35, 36]。
- **まちなかグルメと歴史的カフェ:** 加茂市街地には、造り酒屋の建物をリノベーションし歴史的価値のある空間で料亭の味が楽しめる「Yama Cafe（ヤマカフェ）」 [cite: 37, 38, 39, 40]、町屋造りを活かした日本料理「清雲亭 山重」 [cite: 41, 42, 43]、地元で人気のラーメン店「成龍」や「味一食堂」など、質の高い飲食資源が密集している [cite: 44, 45]。

**回遊ルート構築案：「雪・湯・食の加茂リトリート・ルート」**
滞在時間と消費単価を引き上げるため、以下の回遊ルートを提案する。
午前中、冬鳥越スキーガーデンで雪遊び・ソリ遊びを満喫し、レトロ電車で記念撮影。体が冷えた昼前に移動し、「加茂七谷温泉 美人の湯」で入浴。館内の「ななたに食堂」で地元食材のランチを楽しむ（これでエリアへの飲食・入浴単価が創出される）。午後は加茂市街地（加茂駅周辺）へ移動し、古民家カフェ（Yama Cafe等）でスイーツを楽しみながら、ノスタルジックな加茂の街並みを散策して帰路につく [cite: 32, 38, 46, 47]。

## 6. 最新トレンドの適応可能性

社会の最新トレンドを冬鳥越スキーガーデンに適用し、施設の新しい価値を創造する。

- **インバウンド:** 
本格的な滑走を求めるインバウンドには不向きだが、「初めて雪に触れる」台湾・東南アジアのファミリー層にとっては絶好の施設である。レンタカー移動をする訪日客に対し、「日本の古い木造電車と雪景色」というノスタルジックな風景は、独自のSNS映えスポットとして強力に訴求できる [cite: 6, 13]。

- **通年化:**
現状でもバラ園として機能しているが [cite: 7]、グリーンシーズンのポテンシャルはさらに高い。市はすでにガバメントクラウドファンディング（※ふるさと納税の仕組みを活用し、自治体が特定のプロジェクトの資金を募るクラウドファンディング）を活用し、有名園芸デザイナーを起用した花壇整備などで活性化を図っている [cite: 48]。広大な芝生エリアを活用し、デイキャンプ等の通年利用モデルが期待できる。

- **新規軸（無人化・DXなど）:**
DX（※Digital Transformation：デジタル技術を活用して業務プロセスや顧客体験を変革すること）推進による無人化決済システムの導入が推奨される。
*（※事例：新潟県のGALA湯沢スキー場では、株式会社TOUCH TO GOが提供する次世代無人オーダー決済端末「TTG-MONSTAR」をレンタルや飲食店に導入し、非対面でのサイズ選択から決済までを完全無人化することで、大幅な省人化と顧客満足度向上を実現している [cite: 49, 50]。）*
冬鳥越においても、ロッジでの無人カフェマシンやQRコード決済によるオリジナルグッズの無人販売所の設置など、人件費を抑えつつ客単価を上げる施策が有効である。

## 結論：売上倍増に向けたアクションプラン

冬鳥越スキーガーデンが持つ「低価格」「ソリ特化」「レトロ電車」「美しいガーデン」という独自性を武器に、単体の売上向上だけでなく加茂市全体への経済波及効果を生み出すための施策を提案する。

**短期アクション（1年以内で実行可能・即効性重視）**
