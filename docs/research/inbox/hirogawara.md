# 京都広河原スキー場 集客・売上倍増戦略レポート

## Executive Summary
本レポートは、京都市内で唯一の天然雪を誇る「京都広河原スキー場」の売上倍増に向けた戦略的洞察を提供するものです。当施設は近年、深刻な雪不足やリソースの問題からリフト営業を休止するシーズンが続いていますが、これを「スキー場としての衰退」ではなく「インバウンド向けスノーパークへの業態転換の好機」と捉えることが重要であると考えられます。現状として、道幅の狭い峠道というアクセス面の致命的課題とリフト休止の制約がありますが、一方で「京都市内」という絶大な立地ブランドとプライベート感を併せ持ちます。本リサーチの結論として、既存の本格スキーヤー向け営業から脱却し、インバウンド家族層向けの「雪遊び専用プレミアムパーク」へシフトすべきです。具体的には、鞍馬駅からの完全予約制シャトルバス運行によるアクセス課題の解消、周辺観光資源（くらま温泉や貴船神社など）を組み合わせた高単価な回遊ルートの構築、そしてスノーエスカレーター等の斜面移動ロジスティクスの導入により、高単価・高付加価値な体験型リゾートへの脱皮に向けた具体策を提示します。

## 1. 基礎データと現状のポジショニング

京都広河原スキー場は、豪雪地帯である丹波高地に位置し、京都市街地から車で約90分という立地にある、京都市内唯一のスキー場です [cite: 1, 2]。1990年代前半（約30年前）に開業した当時は、冬季の営業日数が75日〜90日程度確保できており、地元のスキーヤーで大いに賑わっていました [cite: 3]。しかし、近年は暖冬の影響を直接受けやすく、積雪が不安定になり臨時休業が多発する状況に至っています [cite: 3]。まずは施設の基本スペックと現在の市場における立ち位置を整理します。

*   **所在地・連絡先**: 〒601-1101 京都府京都市左京区広河原尾花町286 [cite: 1, 2]
*   **Google MAP**: [https://maps.google.com/?q=35.26667,135.74028](https://maps.google.com/?q=35.26667,135.74028)
*   **電話番号**: 075-746-0850 [cite: 2]
*   **基本規模**: ゲレンデ面積約2ヘクタール、トップ標高635m、ベース標高500m（標高差135m）、最長滑走距離700m [cite: 1, 2, 4]。
*   **コース構成（全1コース・機能的に2分割）**:
    *   **山頂コース**: 全長450m・平均斜度15度。山の斜面を蛇行する林間コースで、すり鉢状の地形も存在する初中級向け [cite: 1, 2]。
    *   **中間コース**: 全長250m・平均斜度12〜13度。直線的で幅が広く滑りやすい初級向けメインゲレンデ [cite: 1, 2]。
    *   **雪遊び広場**: 滑走距離50m・平均斜度8度。ソリ遊びなどに適したエリア [cite: 2, 4]。
*   **主要設備（リフト）**: ペアリフト1基（全長354m）。乗り場が1ヶ所、降り場が中間と山頂の2ヶ所設けられている単線固定循環式特殊索道（一本のワイヤーロープが一定方向に循環し、椅子などの搬器がワイヤーに固定されたまま動く一般的なリフトのこと）です [cite: 1, 4, 5]。
*   **料金体系（通常営業時目安）**: 入場料（大人1,000円、小人500円）、1日券（大人3,000円、小人2,000円）、ナイター券2,500円、駐車場無料（30台） [cite: 1, 4, 5, 6]。※料金は過去シーズンの基準。

これらの基礎データから明らかなように、当施設は極めて小規模なローカルゲレンデです。2021-2022シーズン以降はリフト営業を休止し、主に雪遊び目的の施設として営業（または休業）している状況にあります [cite: 2, 7, 8]。しかし、「京都市内（Kyoto City）」という世界的な観光ブランドエリア内に存在する唯一の天然雪スポットであるという事実は、訪日外国人（インバウンド）市場において計り知れないポテンシャルを秘めています。現在の立ち位置は「地元民の足慣らしゲレンデ」ですが、今後は「インバウンド向けのプレミアム・スノープレイパーク」へとポジショニングを劇的にシフトさせる必要があります。

## 2. 強みと弱み (内部環境)

売上倍増戦略を構築するにあたり、京都広河原スキー場が内包する内部環境のストロングポイントとボトルネックを明確に言語化します。

**強み: 「京都ブランド」と「アットホームなプライベート感」**
最大の武器は、世界的な観光都市「京都」の市内にありながら、大自然の天然雪（トップシーズンには約50センチ〜1メートル前後の積雪が期待できる）に触れられる独自のロケーションです [cite: 1, 2]。小規模であるがゆえに全体が見渡せるすり鉢状のゲレンデは、迷子のリスクが低く、ファミリー層にとって極めて安全で安心な環境を提供します [cite: 1, 7, 9]。また、知名度が低く「穴場」となっているため、土日であっても混雑せず、プライベートゲレンデのような贅沢な空間を独占できる点も、喧騒を逃れたいインバウンド富裕層にとって魅力的な要素となります [cite: 1, 7]。

**弱み: 致命的な「アクセスの難所」と「不安定な雪質」**
一方で、顧客体験を著しく損ねている要因がアクセス環境です。鞍馬方面から花背峠を越えるルート（国道477号・府道38号）は、道幅が狭く（幅員4m未満の1車線〜1.5車線ですれ違いが困難な箇所が連続し）、冬季は圧雪アイスバーン化する上に携帯電話の電波も圏外となる過酷な峠道です [cite: 5, 9]。スタックする車両が続出するなど、雪道運転に不慣れな観光客には自力での到達が極めて困難な状況にあります [cite: 5, 9]。また、暖冬の影響を直接受けやすく、シーズンを通した安定的な積雪が見込めない（臨時休業が多発する）点や、現在リフトが実質休止状態にあるなどのリソース不足も深刻なボトルネックです [cite: 2, 8, 10]。

このように、当施設は「到達難易度の高さ」という致命的な弱みを抱えていますが、これを逆手に取れば「限られた人しかたどり着けない秘境」というブランディングに転換することが可能です。送迎体制さえ構築できれば、弱みはそのまま「希少性（エクスクルーシブ）」という強みに昇華します。

## 3. 競合比較

京都広河原スキー場を、関西圏で集客力を持つ主要な競合施設と比較し、優位性と劣後要因を明確にします。比較対象としては、関西圏の代表格である「びわ湖バレイ（滋賀県）」と「奥伊吹スノーリゾート（滋賀県）」を挙げます。

びわ湖バレイは、人工降雪機を完備した安定したゲレンデ環境と、琵琶湖を一望できる絶景テラスを武器に、インバウンドや若年層から絶大な支持を得ています。奥伊吹は関西最大級の規模と最新設備を備え、日帰りスキーヤーを大量に集客しています。これらの施設に対する京都広河原スキー場の劣後要因は明らかであり、コース規模、雪の安定性、設備の近代化、アクセスの容易さのすべてにおいて圧倒的な差をつけられています。

しかし、京都広河原スキー場には明確な優位性が存在します。それは「京都市内（Kyoto City）」という圧倒的なブランド力と、「混雑のなさ」です。びわ湖バレイなどは週末ともなれば入場制限がかかるほどの混雑を見せますが、広河原は「プライベート感」を保持しています [cite: 1, 7]。したがって、スキー・スノーボードをガッツリ滑り込みたい層をターゲットにするのではなく、「京都観光のついでに、静かな環境で日本の雪に触れてみたい、ソリ遊びを楽しみたい」というニッチなインバウンド層にターゲットを絞り込むことで、競合と全く異なる土俵で戦うことが可能になります。

## 4. ターゲット層のインサイト（口コミ分析）

SNSやレビューサイトに寄せられた実際の顧客（主にファミリー層）の声を分析することで、顕在化している評価と隠れたニーズを浮き彫りにします。

**口コミから見えるポジティブ・ネガティブな傾向**
ポジティブな声としては、「京都市内にスキー場があるなんて信じられない」という驚きや、「こぢんまりとしたサイズ感が子連れにちょうど良く、はぐれる心配がない」といった安全性を高く評価する意見が目立ちます [cite: 7, 9, 11]。また、暖炉のある山小屋風のロッジが「居心地がよい」と空間そのものを楽しむ層も存在します [cite: 9]。
一方でネガティブな声の大部分は、「アクセス道（花背峠）がハードすぎる」「道中でスタックして帰れるか不安になった」といった、アクセスの恐怖に関するものです [cite: 5, 9]。また、リフト券とは別に入場料（1,000円）が徴収されるシステムに対する割高感を指摘する声も見受けられます [cite: 5, 12]。

これらのインサイトから導き出されるのは、「雪遊びの体験価値」自体は高いものの、「そこへ至るまでのアクセス体験」が顧客満足度を大きく下げているという事実です。裏を返せば、マイカーでのアクセスを推奨せず、安全な送迎プラン（シャトルバス等）をセットにして提供できれば、顧客の不満を払拭し、隠れたニーズである「ストレスフリーな雪山体験」を満たすことができます。

## 5. 周辺観光資源とのシナジー（エリア回遊性）

京都広河原スキー場の最大の強みは、京都市内の世界的に有名な観光スポットと連続した動線上に位置していることです。エリア全体の消費単価を上げるための「回遊ルート」を構築します。

**有望な観光資源と回遊ルート構築案**
ゲレンデから京都市街地へ向かうルート上には、「鞍馬寺」や「貴船神社」（スキー場から南へ約25km、車で約45分〜60分の距離）といった日本屈指のパワースポットや、ミシュラン星獲得の摘草料理で知られる花背の「美山荘」（スキー場から南へ約10km、車で約15分〜20分の距離）が存在します [cite: 13, 14, 15, 16, 17, 18, 19]。さらに特筆すべきは、台風等の被害で長期休業していた「くらま温泉」（スキー場から南へ約25km、車で約45分の距離）が、クラウドファンディングを経て2024年末にグランドオープン（再開）した点です [cite: 20, 21, 22]。

これらを繋ぐゴールデンルートとして、**「京都・冬の秘境 スノー＆リトリート・ルート」**を提案します。
午前中は叡山電鉄で鞍馬駅へ向かい、そこからスキー場の送迎バンで広河原へ移動。数時間の「プライベート雪遊び（ソリ・雪だるま作り）」を楽しんだ後、午後に下山して「くらま温泉」で冷えた身体を温め、夕暮れ時の貴船神社（ライトアップ）を参拝。最後に「右源太」や「鳥居茶屋」などの名店で冬季限定の「ぼたん鍋（猪鍋）」を堪能する、という高単価な日帰り、もしくは宿泊パッケージです。このルートを構築することで、単なるスキー場から「京都の冬の総合エンターテインメント」への昇華が可能になります。

## 6. 最新トレンドの適応可能性

売上を飛躍的に向上させるためには、既存の枠組みにとらわれず、最新の業界トレンドを積極的に取り入れる必要があります。

*   **インバウンド（訪日外国人客）**: 東南アジアやオーストラリアからの観光客にとって、「Kyoto」と「Snow」の組み合わせはキラーコンテンツです。本格的なスキー滑走ではなく、「初めて雪に触れる体験（Snow Play）」への需要が爆発的に高まっています。リフトを無理に稼働させずとも、スノーチュービングやかまくら体験、雪上での抹茶提供など、日本文化を掛け合わせた高単価アクティビティを用意することで、多言語対応LPを通じて大きな集客が見込めます。
    *   **ケーススタディ**: この戦略を裏付ける実例として、長野県の「鹿島槍スキー場」は遊園地と協業して「冬のテーマパーク化」を宣言し、ファミリー専用ゲレンデや室内プレイランドを整備することでノンスキーヤー（雪遊び客）の開拓に成功しています [cite: 23]。また、「軽井沢スノーパーク」や「たんばらスキーパーク」などでも、スノーエスカレーターを設置した雪遊び専用エリア（キッズパーク）を整備することで高収益化を実現しており、広河原スキー場においても十分に再現可能なモデルです [cite: 24, 25]。
*   **通年化（グリーンシーズンの活用）**: 広河原エリアは、夏場には「松上げ」という伝統的な火祭りが行われる歴史的背景を持ちます [cite: 26, 27]。松上げとは、毎年8月24日に開催される愛宕神社の火伏せの行事で、高さ約20mの檜丸太の先にある大傘に向かって無数の松明を投げ入れる火祭りです [cite: 28]。シンプルに言えば「巨大な火の玉入れ」あるいは「山間部版の五山送り火」のような壮観かつ幻想的な儀式と言えます [cite: 28]。この地域固有の文化体験と夜の星空観察がメインのグランピング施設を掛け合わせることで、夏場のインバウンド向け高付加価値エコツアーの目玉コンテンツとして直結させることができます。また、林間コースを活かしたマウンテンバイク（MTB）のトレイルランニングコースとして施設を開放し、通年での収益基盤を構築すべきです。
*   **新規軸（DX・無人化）**: リソース不足を補うため、事前決済型のオンラインチケットシステム（QRコード入場）を導入し、現場のスタッフを最小限に抑止します。また、現地のリアルタイムな積雪・道路状況を多言語のAIチャットボットで提供することで、インバウンド客の不安（アクセスや雪不足）を取り除くDX（デジタルトランスフォーメーション）施策が有効です。

## 結論：売上倍増に向けたアクションプラン

リフト営業が休止している現状の制約を逆手に取り、「リフト維持費などの莫大な固定費を削減し、高単価な雪遊び体験を提供する」というビジネスモデルの転換こそが、売上倍増の最適解です。

*   **短期アクション（1年以内で実行可能・即効性重視）**
    1.  **「Kyoto Snow Park」へのリブランディングと多言語LP展開**: スキーヤー向けのアピールを停止し、インバウンド家族層向けの「雪遊び専用プレミアムパーク」として多言語のランディングページを構築する。ソリ、スノーチューブ、長靴のフルセットレンタルを高単価（例: 8,000円〜）で提供する。
    2.  **鞍馬駅〜スキー場間の公式シャトルバス運行（完全予約制）**: マイカーでのアクセスを制限・非推奨とし、雪道に強い四輪駆動バンを用いた鞍馬駅からの送迎サービスをパッケージ化する。これにより、最大の弱点である「道中の恐怖」を解消し、集客のボトルネックを破壊する。
    3.  **斜面移動のロジスティクス確保**: ソリ遊び等を提供する場合、「斜度15度前後の斜面を滑り降りた後、顧客はどのようにしてスタート地点へ登り返すのか？」という課題が生じます。これに対し、スノーエスカレーター（マジックカーペット型の動く歩道）を新設する投資を行うか、あるいはスノーモービルによるソリの牽引（リフトアップ）サービスを導入し、顧客が歩いて登り返す苦労をゼロにする斜面移動のロジスティクス確保が必須となります。
    4.  **「くらま温泉」との連携日帰りチケットの販売**: 復活したばかりの「くらま温泉」と提携し、「広河原で雪遊び＋くらま温泉入浴」のジョイントチケットをオンライン販売し、京都観光の新たな定番冬ルートとしてOTA（オンライン旅行代理店）に掲載する。
