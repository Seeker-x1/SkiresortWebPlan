# カムイスキーリンクス 売上倍増戦略リサーチレポート

**JAPOW ID: 37**

## エグゼクティブサマリ

カムイスキーリンクスは、北海道旭川市近郊に位置し、道北最大級のスケールと極上のパウダースノーを有するスノーリゾートである。本レポートは、同施設の現状とポテンシャルを多角的に分析し、滞在時間の延長およびエリア全体の消費単価向上による売上倍増に向けた具体的なアクションプランを提供するものである。なお、本レポートは観光マーケティングに基づく戦略的情報提供を目的としており、専門的な財務・投資助言を構成するものではない。

本施設の最大の特徴は、市街地からの圧倒的なアクセスの良さと、非圧雪エリアの充実度にある。一方で、リゾート型インフラ（宿泊・温泉等）の欠如や、旧式リフトによる厳冬期の防寒性不足が大きな課題となっている。また、燃料費高騰等の影響により最大80％の段階的値上げが計画されており、これまで最大の武器であった「価格優位性」が失われる過渡期にある。

売上を倍増させるための最終結論として、施設単体の魅力向上だけでなく、旭川市街地の観光・飲食資源と結びついた「都市観光型スノーリゾート」としてのブランド再構築を推奨する。短期的な即効性のある施策として「プレミアム・アーリーライド（早朝貸切）とサウナ・グルメ周遊パスの展開」、中長期的には「公的資金や民間活力（PFI）を活用したフード付きリフトへの更新」および「インバウンドとローカル客の二重価格戦略の確立」を提言する。

## 1. 基礎データと現状のポジショニング

カムイスキーリンクスは、優れたアクセスと本格的なコースレイアウトを両立させた類稀な公設スキー場である。

- **歴史的背景と運営体制:** 
  昭和初期に「国設神居山スキー場」として開設され、1984年に現在の規模へ拡大された。かつての運営企業（日本ゴルフ振興）の経営破綻に伴い、2003年に旭川市が施設を無償譲受し、存続を決定した公的な成り立ちを持つ [cite: 1, 2]。現在は市が所有し、「一般社団法人大雪カムイミンタラDMO」が指定管理者として管理・運営を担っている [cite: 2, 3]。
- **基本情報とアクセス:** 
  旭川市内中心部から車で約30分、就航率99％を誇る旭川空港から約60分という圧倒的な好立地にある [cite: 4, 5]。
  - **現地住所:** 〒074-1181 北海道旭川市神居町西丘112 [cite: 2, 6, 7]
  - **連絡先 (TEL):** 0166-72-2311 [cite: 7, 8]
  - **Google MAP:** [マップを開く](https://www.google.com/maps/search/?api=1&query=カムイスキーリンクス)
- **規模とコース構成:** 
  総面積は166ha、標高差は601m（山頂751m〜山麓150m）におよぶ [cite: 2, 4, 9]。新規解放を含む全26コースを備え、初級35%、中級30%、上級35%とバランスの良い斜面構成を持つ [cite: 4, 8, 9, 10]。最長滑走距離は4,000m、最大斜度は35度に達する [cite: 4, 11]。FIS（国際スキー連盟）公認の競技用バーンを有するほか、コース全体の約35%が未圧雪のパウダーエリアとして開放されている [cite: 9, 10]。
  **【主要コースの詳細】** [cite: 12, 13]
  - **ゴールドコース（1・2・3）:** 長さ3,000m / 最大斜度30度（中級）。広大なカムイのメインコース。
  - **白樺コース（1・2）:** 長さ1,800m / 最大斜度28度（中級）。競技向けにデザインされた変化に富んだコース。
  - **クリスタルコース:** 長さ3,500m / 最大斜度18度（初級）。樹氷に囲まれた美しいロングランコース。
  - **エゾマツコース:** 長さ800m / 最大斜度35度（上級）。夏場の沢地形を活かした自然のハーフパイプ状コース。
  - **ダイナミックコース:** 長さ1,100m / 最大斜度30度（上級）。北斜面に位置する豪快なコース。
  - **深雪コース:** 長さ1,000m / 最大斜度35度（上級）。極上のパウダースノーが堪能できる専用斜面。
  - **シルキーコース:** 長さ800m / 最大斜度18度（上級）。全面コブの名物バーン。
  - **ファミリーコース:** 長さ900m / 最大斜度18度（初級）。センターハウス前に広がる緩斜面。
- **リフト・ゴンドラ設備:** 
  計6基の索道設備を備え、黄色のゴンドラが施設のアイコンとなっている [cite: 9, 10, 13, 14]。
  - **カムイゴンドラ:** 2,327m（メインの輸送力を誇る4人乗り箱型ゴンドラ）
  - **第1ペアリフト:** 733m
  - **第2ペアリフト:** 973m
  - **第3ペアリフト:** 638m（近年改修済）
  - **第4ペアリフト:** 754m
  - **第5ペアリフト:** 999m
- **料金体系と今後の価格戦略:** 
  2024-2025シーズンの大人1日券は5,300円である [cite: 8, 15]。しかし、燃料費や電気料金等の施設維持費高騰に対応するため、条例改定に基づく段階的な値上げを実施中であり、現行水準から約80%増となる上限（約9,500円）への将来的な料金引き上げが計画されている [cite: 16, 17]。この値上げにより「低価格」という優位性は失われるが、代替措置として市民や近隣住民向けに使い勝手の良い「20時間券」を新たに導入し、ローカル客とインバウンド客に対する実質的な「二重価格（プライシング）戦略」を展開することで地元客の離反を防ぐ構造となっている [cite: 16]。

## 2. 強みと弱み (内部環境)

施設の評価を二分する要因を内部環境の視点から紐解く。

### 内部環境の強み (Strengths)
- **究極の雪質 (シルキースノー):** 内陸部特有の厳しい冷え込みと乾燥した気候により、水分量の極めて少ないサラサラのパウダースノーが高確率で提供される [cite: 4, 6, 18]。
- **未圧雪・ツリーラン（樹林帯の中を滑走するバックカントリー的要素を持つ滑走スタイル）エリアの充実:** 上級者コースの大部分が非圧雪であり、地形の変化に富んだ800〜1200mのロングコースや、森の中を滑り抜ける専用エリアが豊富に設定されている [cite: 10, 18]。
- **ゴンドラの輸送力と混雑の少なさ:** 大規模スキー場でありながら、本州のツアー客が比較的少ないためリフト待ち時間がほぼゼロであり、滑走効率が非常に高い [cite: 14, 18]。
- **山頂レストランの魅力:** 標高751mの絶景とともに本格的なイタリアンピッツァやパスタを提供する「Trattoria & Pizzeria Monte751」が、施設価値を高めている [cite: 15, 19]。さらに山麓の「らぁめん食堂 NOBu リンクス店」では本格的な旭川ラーメンが提供され、ゲレンデ食の枠を超えた評価を得ている [cite: 20, 21]。

### 内部環境の弱み (Weaknesses)
- **リフトの防寒性の欠如:** ゴンドラ以外のリフト（ペアリフト）にはフードがなく、氷点下10度を下回る厳冬期には移動中の冷え込みが過酷となるため、厳重な防寒対策が強いられる [cite: 14]。
- **積雪量の変動リスク:** 雪の質は最高峰である一方、「どか雪」が降ることは稀なため、タイミングによっては深い新雪（ディープパウダー）が味わえない日がある [cite: 14]。
- **リゾート的インフラの不足:** ゲレンデサイドに宿泊施設や温泉、大型エンタメ施設が存在せず、滞在してアフタースキーを楽しむ仕組みがない。また、山麓のセンターハウスは古く手狭である [cite: 14]。
- **二次交通のボトルネック:** 旭川市街地からのアクセスを路線バスに依存しており、インバウンド増加に伴う朝イチ（往路第1便）の混雑が大きな課題となっている [cite: 9]。

## 3. 競合比較

道内の同規模大型リゾートとの相対的な比較により、マーケティングの方向性と価格戦略のシフトを明確にする。道央・道北エリアで滑走距離や規模感が類似する主たる競合施設として、「星野リゾート トマム」および「富良野スキー場」が挙げられる [cite: 16, 22, 23]。

**競合に対する優位性と劣後要因の変遷:**
現在の最大の優位性は「コストパフォーマンスの高さ」と「ローカル感」である [cite: 16, 26]。競合の1日券が7,500円〜8,000円であるのに対し、本施設は5,300円と割安である。しかし、将来的な料金引き上げ（上限約9,500円）が実行されれば、この価格優位性は完全に失われる [cite: 16, 17]。
したがって、今後は「安いから選ばれる」のではなく、旭川市という巨大な都市リソース（ローカルグルメ・サウナ・ホテル）とシームレスに連動した**「都市観光機能拡張型リゾート」**としての絶対的な価値へとシフトする必要がある。競合施設がリゾート内に完結する「箱庭型」であるのに対し、本施設は街全体をアフタースキーの舞台として活用できる点が、今後の最大の武器となる [cite: 16, 27]。

## 4. ターゲット層のインサイト（口コミ分析）

SNSやレビューサイトから、利用者の期待値と隠れた不満を分析する。

- **ポジティブな傾向:** 「日本最大のローカルスキー場」「富良野に匹敵する広さ」「コース外滑走（ツリーラン）が最高」など、ゲレンデのスケールと非圧雪エリアの自由度への絶賛が多い [cite: 14, 18]。また、ゲレンデ食としては異例とも言える「ラーメン（らぁめん食堂NOBu）が本当に美味しい」という食への評価も見受けられる [cite: 14, 28, 29]。ニセコ等に行かなくとも極上のパウダーが楽しめる穴場として支持されている。
- **ネガティブな傾向とペイン（顧客が体験する悩みや不満点）:** 「とにかく寒い。フード付きリフトが1本もないのは厳しい」という防寒設備への強いペインが散見される [cite: 14]。また、「積雪量が少なくパウダーがない日もある」という期待値のズレや、交通手段が旭川市内からのバスに限られ、特に朝のピーク時に乗車できない事態が発生する不便さも指摘されている [cite: 9, 14]。

## 5. 周辺観光資源とのシナジー（エリア回遊性）

ゲレンデ内に宿泊設備がない弱みを逆手に取り、旭川市街地の強力な観光資源と連携した「回遊ルート」を構築することが不可欠である。

- **近隣の有望な観光資源:** 
  旭川市中心部（さんろく街周辺）には、「旭川らぅめん青葉 本店」や「みそラーメンよし乃本店」などの有名ラーメン店、「成吉思汗 大黒屋 5丁目支店」「炉端のユーカラ」といった北海道グルメの名店がひしめいている [cite: 30, 31, 32, 33, 34]。また、冷えた体を癒やす施設として、「SPA&SAUNA オスパー」や高砂台の「旭川高砂台 万葉の湯」、深川の「イルムの湯 アグリ工房まあぶ」など、サウナ・温浴施設も豊富である [cite: 35, 36, 37]。
- **回遊ルート構築案:** 
  「カムイ極寒パウダー × 旭川熱波サウナ × ローカルディープグルメ」という回遊ルートを推進する。日中はカムイの乾いた雪とツリーランを満喫し、夕方には市街地に戻る。オスパー等の施設でサウナに入り体を整えた後、さんろく街でジンギスカンとクラフトビールを堪能する。これにより、ゲレンデ単体の消費からエリア全体での高単価消費へと誘導できる [cite: 27, 38]。

## 6. 最新トレンドの適応可能性

社会的なトレンドを取り入れることで、新規顧客の開拓と単価向上が見込める。

- **インバウンドの取り込み:** 現状は東アジアからの日帰り層が中心だが、長期滞在し消費額の高い欧米豪のパウダー愛好家をターゲットにする余地が十分にある [cite: 9, 39]。彼らに刺さる深雪体験やローカルガイドツアーを提供し、富裕層向けの単価引き上げを狙う [cite: 39]。
- **通年化の推進:** 秋期にはゴンドラを利用した紅葉観賞「カムイゴンドライド」が稼働している [cite: 2, 40]。今後は、電動マウンテンバイク（e-MTB）の常設コース化など、グリーンシーズンにおける本格的な収益事業の構築が求められる [cite: 39]。
- **新規軸（DX化とワーケーション）:** DX化（デジタル技術を活用してビジネスプロセスや顧客体験を変革すること）が進行しており、市内中心部のホテル（OMO7旭川など）にリフト券の自動発券機を設置し、Web購入から現地発券までをスムーズにしている [cite: 10]。また、施設内の飲食店に自動食券機を導入しキャッシュレス化も進んでいる [cite: 10]。ワーケーションプランと連動し、「午前中は仕事、午後はカムイでパウダー」という新しいライフスタイルの提案も有効である [cite: 38]。

## 結論：売上倍増に向けたアクションプラン

上記のリサーチに基づき、値上げ後の価格競争力低下を補い、顧客単価を飛躍させるための現実的な施策を提案する。

- **短期アクション（1年以内で実行可能・即効性重視）**
  - **プレミアム・アーリーライド＆ローカルガイドの創設:** 欧米豪の富裕層やコアスキーヤー向けに、通常営業前のゴンドラ優先乗車と、未圧雪・ツリーランエリアの安全なルーティングをセットにした高単価ガイドツアーを販売する。
  - **「スノー＆サウナ・グルメ」デジタルパスの発行:** 旭川市街地のホテル、サウナ（オスパー等）、協賛飲食店と連携し、リフト券と市街地での温浴・飲食に使えるバウチャーをセットにしたWeb限定パスを販売。インバウンドを中心とした市街地への回遊を強固にする。
  - **アクセスバスの事前予約制導入と増便:** 朝一番のバス混雑を解消するため、市内ホテルを巡回する事前予約・決済制のシャトルバスを新設し、インバウンド客の移動ペインを取り除く [cite: 9]。

- **中長期アクション（1〜3年スパン・根本的な価値向上）**
  - **ローカル客とインバウンド客の二重価格戦略の確立:** リフト券の定価を大きく引き上げる一方で、新たに導入予定の「20時間券」や「市民割」を徹底活用し、地元客の離反を防ぐ。同時に、インバウンド向けにはファストパス（優先搭乗）を付与した高単価チケットを販売し、収益の最大化を図る [cite: 16]。
  - **官民連携によるフード付き高速リフトへの更新投資:** 顧客最大の不満である「リフトの寒さ」を根本から解決するため、主要なアクセスルートとなるペアリフトをフード付きクワッドリフトへ更新する。旭川市所有の公的施設であることを活かし、国の観光庁補助金や地方創生交付金の獲得、またはPFI（民間資金等活用事業）やネーミングライツの導入など、民間活力を活用したスキームで数十億円規模の投資資金調達ロジックを構築する [cite: 1, 2]。
  - **グリーンシーズン・アクティビティの本格事業化:** 広大な斜面とゴンドラ設備を活かし、世界基準のマウンテンバイクトレイル（MTB）や、絶景を楽しめる高規格なデイキャンプ/グランピングエリアを常設し、通年での安定した収益基盤を確立する。
